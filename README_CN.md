# 易之书

- [English Version](README.md)

《周易》是一部古老而又灿烂的文化瑰宝，
古人用它来预测未来、决策国家大事、反映当前现象。

古人把三种爻组合在一起变成卦，卦的组合有八种，所以又称为《八卦》。

《八卦》的八种物质代表，
天[乾]、地[坤]、雷[震]、风[巽]、水[坎]、火[离]、山[艮]、泽[兑]。

《易》中的八经卦﹐两两重复排列为六十四卦。

- **乾**为天（第1卦）：源动力刚健，利于开创伟业。
- **坤**为地（第2卦）：宽宏包容力强，利于生养万物。
- 水雷**屯**（第3卦）：有艰难险阻，宜于屯聚力量。
- 山水**蒙**（第4卦）：浑沌初开，适宜充实学养。
- 水天**需**（第5卦）：时机未到，要耐心等待。
- 天水**讼**（第6卦）：碰到口舌是非。
- 地水**师**（第7卦）：以正压邪，率众讨伐。
- 水地**比**（第8卦）：随和，亲善，与人和睦相处。
- 风天**小畜**（第9卦）：积累尚浅，不够充分。
- 天泽**履**（第10卦）：大力践行，礼节行事。
- 地天**泰**（第11卦）：通泰，通畅，诸事恒通。
- 天地**否**（第12卦）：受阻，闭塞，无能为力。
- 天火**同人**（第13卦）：随大流，从众为要。
- 火天**大有**（第14卦）：喜获丰收。
- 地山**谦**（第15卦）：谦逊谨慎，深藏不露。
- 雷地**豫**（第16卦）：欢快愉悦，心畅事宜。
- 泽雷**随**（第17卦）：步人后尘，归顺强者。
- 山风**蛊**（第18卦）：拨乱反正，不避亲贤。
- 地泽**临**（第19卦）：贵人莅临，高人惠顾。
- 风地**观**（第20卦）：考察，观望，三思而后行。
- 火雷**噬嗑**（第21卦）：坚持沟通，有效交流。
- 山火**贲**（第22卦）：内重礼仪，外树形象。
- 山地**剥**（第23卦）：去伪存真。
- 地雷**复**（第24卦）：迷途知返。
- 天雷**无妄**（第25卦）：无妄为，无过失。
- 山天**大畜**（第26卦）：积畜已久，准备充分。
- 山雷**颐**（第27卦）：颐养身心。
- 泽风**大过**（第28卦）：太过分，已遭损失。
- **坎**为水（第29卦）：坎险拦路。
- **离**为火（第30卦）：外露，炫耀，宜收敛。
- 泽山**咸**（第31卦）：心有灵犀，感应灵通。
- 雷风**恒**（第32卦）：恒久不变。
- 天山**遯**（第33卦）：逃遁，隐退。
- 雷天**大壮**（第34卦）：众目睽睽，招风惹火。
- 火地**晋**（第35卦）：晋升，前进。
- 地火**明夷**（第36卦）：老道沉稳，自觉规避风险。
- 风火**家人**（第37卦）：治家在先。
- 火泽**睽**（第38卦）：反目，背驰。
- 水山**蹇**（第39卦）：艰险，难关。
- 雷水**解**（第40卦）：解开，释放。
- 山泽**损**（第41卦）：损失，受损。
- 风雷**益**（第42卦）：收益，受益。
- 泽天**夬**（第43卦）：有缺陷，不完整。
- 天风**姤**（第44卦）：邂逅，巧合，疏而不密。
- 泽地**萃**（第45卦）：出类拔萃。
- 地风**升**（第46卦）：上升，提升。
- 泽水**困**（第47卦）：受困，被围。
- 水风**井**（第48卦）：规划，约束，坚守。
- 泽火**革**（第49卦）：改变，革新。
- 火风**鼎**（第50卦）：独当一面，极致。
- **震**为雷（第51卦）：欲动，发动，行动。
- **艮**为山（第52卦）：壁垒，受阻。
- 风山**渐**（第53卦）：渐进，缓行。
- 雷泽**归妹**（第54卦）：结亲，结缘，缔结亲缘。
- 雷火**丰**（第55卦）：丰富，张扬。
- 火山**旅**（第56卦）：旅行，客居。
- **巽**为风（第57卦）：风行，上行下效 。
- **兑**为泽（第58卦）：谈话，愉悦。
- 风水**涣**（第59卦）：涣散，发散。
- 水泽**节**（第60卦）：节制，暂止。
- 风泽**中孚**（第61卦）：守正，诚信。
- 雷山**小过**（第62卦）：过火，可补救过失。
- 水火**既济**（第63卦）：心想事成，顺畅。
- 火水**未济**（第64卦）：事业未竟，尚需努力。

《易经》蕴含的取舍之道：

1. 乾 ䷀：奋斗信仰的取舍。野心是改变命运的特效药，奋斗目标尽在胸中，奋斗的关键在于行动。
2. 坤 ䷁：器量大小的取舍。宽容助人生一帆风顺。宽容是打开成功之门的金钥匙，要成为宽容的人。
3. 屯 ䷂：打夯深浅的取舍。机遇只垂青有准备的人，认真二字抵挡千军万马。
4. 蒙 ䷃：启蒙之道的取舍。创新“勇”无止境，创新是创造性思维的来源。
5. 需 ䷄：坚忍的取舍。吹尽黄沙始到金，天生我材必有用。
6. 讼 ䷅：争执的取舍。争执有度，妥善处理合作中的争执。误会早解除早安心。
7. 师 ䷆：树立威望的取舍。焊牢与下属的关系，正确处理与下属的冲突，尊重下属才是树立威望的长久之策。
8. 比 ䷇：合作竞争的取舍。合作是团队作战的利器，合作建立在信任之上。
9. 小畜 ䷈：计划的取舍。生活离不开计划，目标要远大，不要畏惧调整计划。
10. 履 ䷉：行动决策的取舍。超前决策更胜一筹，个人如何做决策，团队如何做决策，需要取舍。
11. 泰 ䷊：刚与柔的取舍。做人要有刚有柔，以柔制刚是法宝。
12. 否 ䷋：身处逆境的取舍。逆境是巨大的财富，勇敢的面对逆境，战胜逆境，完成蜕变。
13. 同人 ䷌：凝聚人心的取舍。合作才是最佳方案，搞掂三类人：受害者、悲观者、反对者。
14. 大有 ䷍：博爱的取舍。爱是一种循环，助人即助己。
15. 谦 ䷎：谦卑的取舍。做人谦虚是正道，莫入谦虚的误区。
16. 豫 ䷏：快乐人生的取舍。快乐是什么，莫与快乐擦肩而过。不断学习；关注你在做的事情；好好照顾自己；活在当下；无条件的爱。
17. 随 ䷐：遵从自然的取舍。制定科学的作息表，熬夜时保养有加。
18. 蛊 ䷑：力挽狂澜的取舍。危机制胜是领导者的必备素质，危机的背面是转机。
19. 临 ䷒：督导他人的取舍。辩论无法改变对方，要善为人师，不要好为人师。
20. 观 ䷓：师从他人的取舍。吸取别人的经验才能少走弯路，将别人的优点揽入囊中。
21. 噬嗑 ䷔：制度的取舍。好制度胜过一切，执行制度一蹴而就。
22. 贲 ䷕：思维的取舍。生命不息，学习不止，用思维能力打拼天下。
23. 剥 ䷖：未雨绸缪的取舍。居安思危者方能保其身，深谋远虑是成功的助推器。
24. 复 ䷗：复归的取舍。懂得放弃是大智慧，学会放弃才会成功。
25. 无妄 ䷘：妄为的取舍。约束自己先做正确的事，量力而为，知足常乐。
26. 大畜 ䷙：养精蓄锐的取舍。坚定的脚印，懂得积蓄力量才是人生的强者。
27. 颐 ䷚：颐养的取舍。运动常在，笑颜常在，饮食调养，长命百岁。
28. 大过 ䷛：坚持自我的取舍。将命运紧紧握在手中，拥有独立人格。
29. 坎 ䷜：执着的取舍。一诺千金，坚持下去才有成功的可能。
30. 离 ䷝：借力的取舍。借力才能成大事，生意人的“借”道。
31. 咸 ䷞：虚怀若谷的取舍。闻过则改，消灭两大对立情绪。
32. 恒 ䷟：坚守的取舍。对自己负责，坚守自我。
33. 遯 ䷠：朋友关系的取舍。小人是人格有缺陷的人，不要和小人计较是非成败，傲然之姿，自有震摄之力。
34. 大壮 ䷡：德行的取舍。社会之中以德立身，找到自己的道德底线。
35. 晋 ䷢：才华发挥时的取舍。在潜意识的花园中掘金，让自己在“瞬间”中增值。
36. 明夷 ䷣：韬光养晦的取舍。聪明外露，不如难得糊涂；锋芒毕露，可能招惹祸端。
37. 家人 ䷤：家庭幸福之道的取舍。和睦家庭，忍让第一，批评指责威胁家庭关系。
38. 睽 ䷥：求同存异的取舍。宽以待人，加点人际交往的润滑剂。
39. 蹇 ䷦：寻找贵人的取舍。自省可避免迷失自己，在自省中提高自己。
40. 解 ䷧：危机当下的取舍。培养化解危机的能力，危机之中蕴机遇。
41. 损 ䷨：欲望限度的取舍。欲望是成功的动力，莫要欲壑难填。
42. 益 ䷩：见贤思齐的取舍。砸烂借口的挡箭牌，竖起改过的盾牌，无知胜有知。
43. 夬 ䷪：施恩之道的取舍。施恩也要讲技巧，莫将施恩变施舍。
44. 姤 ䷫：爱情中的取舍。经营爱情就像放风筝，说“不”并不难。
45. 萃 ䷬：编织交际网络的取舍。识人交挚友，不断扩大交际圈，关系网需要精细维护。
46. 升 ䷭：前进方式的取舍。永不停止前进，脚比路长。
47. 困 ䷮：苦难困顿中的取舍。寻找贫困之中的黄金屋，打赢贫困翻身仗。
48. 井 ䷯：授人玫瑰的取舍。生命在于给予，用心助人。
49. 革 ䷰：变革中的取舍。每天改变1％，让自己不断变革。
50. 鼎 ䷱：创新中的取舍。创新第一，自我提问，及时记录下来一些创新想法。
51. 震 ䷲：威武不能屈的取舍。大丈夫的立世之道，绝境不等于失败。
52. 艮 ䷳：契合实际的取舍。平常心最可贵，买来金刚钻，再揽瓷器活。
53. 渐 ䷴：循序渐进的取舍。一步一步走向目标笨鸟先飞，勤可补拙。
54. 归妹 ䷵：家庭和睦的取舍。驾驭婚姻的方舟，不要引爆婆媳关系的定时炸弹。
55. 丰 ䷶：成功之道的取舍。天下大事必做于细，从小起步。
56. 旅 ䷷：漂泊在外的取舍。借梯登天，小人物有大用处。
57. 巽 ䷸：谨慎做事的取舍。严谨立身成大业，慎言修身。
58. 兑 ䷹：挚友互助的取舍。互助的效应，信任是合作之基。
59. 涣 ䷺：收敛人心的取舍。走入人心得人心，管理柔为上。
60. 节 ䷻：节制行为的取舍。冲动是魔鬼，多磨练方自控。
61. 中孚 ䷼：诚信之心的取舍。诚信，立世之基；诚信，趋人而同。
62. 小过 ䷽：纠错态度的取舍。正确的纠错态度，批评也有艺术。
63. 既济 ䷾：坐天下的取舍。后继有人，基业长青，居安思危永葆战斗力。
64. 未济 ䷿：明辨本质的取舍。将偏见关入牢笼，冷静有时是根救命草。

- 当你没钱的时候，把勤舍出去，钱就来了。这叫 **天道酬勤** 。
- 当有钱的时候，把钱舍出去，人就来了。这叫 **轻财聚人** 。
- 当有人的时候，把爱舍出去，事业就来了。这叫 **厚德载物** 。
- 当事业成功后，把智慧舍出去，喜悦就来了。这叫 **德行天下** 。
