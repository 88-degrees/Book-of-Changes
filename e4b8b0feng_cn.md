# 丰 ䷶

<img src="shapes/55.10.jpg" width="101" alt="丰" align="left">

丰（fēng）卦的代号是`5:4`。主卦是`5`卦，离卦，卦象是火，光明而依附，在这个卦的爻辞中，主方被比喻为日食时的太阳。客卦是`4`卦震卦，卦象是雷，运动而乏力。震卦和离卦的下爻和中爻都相同，区别在上爻：离卦的上爻，阳，震卦的上爻，阴。既然主方被比喻为日食中的太阳，客方就可以比喻为日食中的月亮。这个卦的爻辞用全日食过程作比喻，说明主方在客方的激烈挑战前，应当抓住机遇，创造对主方更有利的新局面。

图中，红色表示当位的爻，天蓝色表示不当位的爻，箭头表示有应。

- 卦序：55

> 豐，亨，王假之，勿憂，宜日中。
>《彖》曰：豐，大也。明以動，故豐。王假之，尚大也；勿憂，宜日中，宜照天下也。日中則昃，月盈則食，天地盈虛，與時消息，而況於人乎？況於鬼神乎？
>《象》曰：雷電皆至，豐，君子以折獄致刑。

> 初九，遇其配主，雖旬无咎。往有尚。
>《象》曰：雖旬无咎，過旬災也。

> 六二，豐其蔀，日中見斗，往得疑疾，有孚發若，吉。
>《象》曰：有孚發若，信以發志也。

> 九三，豐其沛，日中見沬，折其右肱，无咎。
>《象》曰：豐其沛，不可大事也；折其右肱，終不可用也。

> 九四，豐其蔀，日中見斗，遇其夷主，吉。
>《象》曰：豐其蔀，位不當也；日中見斗，幽不明也；遇其夷主，吉行也。

> 六五，來章，有慶譽，吉。
>《象》曰：六五之吉，有慶也。

> 上六，豐其屋，蔀其家，闚其戶，闃其无人，三歲不覿，凶。
>《象》曰：豐其屋，天際翔也；闚其戶，闃其无人，自藏也。

### 雷火丰 日中则斜 上上卦

古镜昏暗好几年，一朝磨明似月圆，君子谋事逢此卦，近来运转喜自然。

### 结构和卦爻辞

1. 主方与火很相似，客方与雷很相似。
2. 主方的阳数是`5`，客方的阳数是`4`。
3. 主方的行动是阳，素质是阴，态度是阳；客方的行动、素质和态度也是阳、阴和阴。

丰卦下部的三条爻是主卦，代表主方；上部的三条爻是客卦，代表客方。
黑色的爻是当位的爻，对主方有利；灰色的爻是不当位的爻，对主方不利。

箭头表示主卦和客卦中相应的一对爻是处于阴阳和谐状态，有应。
箭头方向是从阴爻到阳爻。下爻或上爻有应对主方有利，不有应对主方不利。
中爻有应不一定对主方有利，不有应不一定对主方不利，需要结合下爻和上爻作具体分析。

主卦的三条爻全当位，表示主方的状态是恰当的，主方应当坚持目前状态：
积极主动地谋取发展，维持改善素质的需要，对客方的态度强硬以制约客方。
客卦中四阳和五阴不当位，表示客方的积极行动与主方的积极行动有冲突，还表示客方的素质不佳，不能让主方获益。卦的上爻是对中爻的补充。主卦的中爻是阴，上爻是阳，表示主方的素质虽差，但不是很差，力量弱，但不是很弱；客卦的中爻和上爻都是阴，表示客方的素质很差，力量很弱。由此可见，主方的素质优于客方，主方的力量强于客方。综合起来看，主方面临客方的剧烈竞争，主方应当抓住机遇，依靠素质方面的优势，创造更多利益，夺取胜利。

丰卦的卦爻辞用日和月形象地比喻主方和客方。
主方在运动中，带着光明，如同太阳；客方也在运动中，但不发光，如同月亮。
二者在天空中偶尔相遇在与地球的同一条直线上，出现日食。
卦辞结合卦的总体结构，说明当前形势，给于主方策略性的建议。

爻辞实际上是一篇很古老的关于全日食的记载：
```
国王来（观看日食）了，
（日食）一般都发生在太阳当中的时候。
月亮来到太阳的旁边，一明一暗，
与太阳在天空中配对。
虽然已经等了十多天，
再等一会就可以看到罕见的现象。
月亮像个大草席，向太阳盖过来了，
大白天看到了北斗星，天黑得想象不到地快。
太阳被月亮盖了很多，中午看到微暗的天空，
太阳的右胳膊（像是）被折断了。
月亮像个大草席，向太阳盖过来了，
大白天看到了北斗星。
月亮来到太阳的前面，把太阳完全夷平了。
最美丽的时刻到来了，一片美好的赞誉。
（全食以后的瞬间，太阳刚刚微露出一点点，
一股极强的光环显现在太空，
像是一枚灿烂无比的巨大钻石戒指）。
月光盖过了房屋，像大草席覆盖了家，
窥视这户人家，家中寂静无人，三年看不到人。
（爻辞巧妙地从日食转移到了现实中的生活）。
```

### 亨，王假之；勿忧，宜日中。
```
顺利，王来了，不要忧虑，宜于在正是日当中天的时候。
〖解说〗“假”（jiǎ），照规定或请求暂时离开工作、学习场所，如暑假，休假。
主客双方都积极主动，双方的素质都不佳，主方不能指望从客方获益。
主方态度强硬，客方态度随和，主方的事情可以进行得顺利。
尽管成功的机会很少，毕竟有成功的机会，就像是日全食，
日与月亮也会偶然在天空与地球的同一条直线上相遇。
日食是罕见的自然现象，王也来观看了，这种自然现象都在大白天发生。
与日食现象相似，成功的机会也是少有的，需要发现机遇，抓住机遇。
```

### 一阳：遇其配主，虽旬无咎，往有尚。
```
遇到它与主人相配，
虽然已有十日，无所怪罪，往前有高尚的事。
〖解说〗这是第一爻的爻辞，阳，
表示主方积极主动地谋取和扩大自己的利益，
比如说，创新、创业、投资、进攻、求职、示爱，等等。
“旬”（xún），十日为一旬。
“尚”（shàng）是尊崇，高尚。
“配”（pèi），两性结合：配偶，配种。
在当前主客双方关系中，
主方与客方都在为自己的利益而积极主动地行动，
双方在竞争中，如同太阳和月亮在天空中相配成对。
这种日全食现象在初十以后才会发生，耐心等待。
主方也需要耐心地对待客方的挑战，主方无所怪罪。
“遇其配主”只是日食刚刚开始，往前发展，景观更壮丽。
对于主方来说，客方的挑战刚刚开始，
主方须要准备应对更严重的挑战。

〖结构分析〗这条爻是阳爻在阳位，
当位，然而与四阳不有应。
当位表明主方积极主动，有可能取得更多利益，
是潜在的对主方有利的因素；
不有应表明客方也是积极主动，
主方遇到强烈的竞争，
这潜在因素没有成为真正的对主方有利的因素。
所以爻辞只是说“无咎”，“ 往有尚”。
主方应当坚持积极主动，继续努力。
```

### 二阴：丰其蔀，日中见斗，往得疑疾；有孚发若，吉。
```
大部分被盖了，中午看到了北斗星，
往前快得想象不到；有诚信就好有发展，吉利。
〖解说〗这是第二爻的爻辞，
阴，表示主方的素质不佳。
“蔀”（bù），覆盖于棚架上以遮蔽阳光的草席。
“疾”（jí），快，迅速：疾步。
“发”（fā），开展，张大，扩大：发展。
“若”（ruò），好像：旁若无人。
爻辞的意思是：太阳被月亮如同大草席般地覆盖，
日在中天的时候看到了北斗星，
进展得想象不到地快；有诚信就会有发展，吉利。
主方在谋求发展，由于素质不佳，
一时遏制不住客方的进攻，如同太阳在日食时失去光辉，
这种趋势的发展非常快，意想不到地快。
在这种情况下，须要有诚信，有诚信就会有发展，对主方吉利。

〖结构分析〗这条爻是阴爻在阴位，
当位，然而与五阴不有应。
当位表明主方的素质不佳是谋求发展的动力，
是潜在的对主方有利的因素；
不有应表明客方的素质也不佳，也在积极谋求利益，
这潜在因素没有成为真正的对主方有利的因素。
由于存在这种潜在的对主方有利的因素，
如果主方诚恳地对待客方，取得客方信任，做到互动互利，
这对主方有好处，所以爻辞说“吉”。
“吉”是有条件的，必须是“有孚发若”。
```

### 三阳：丰其沛，日中见沬，折其右肱，无咎。
```
它被覆盖了很多，
中午看到了微暗的天空，
它的右胳膊被折断了，无所怪罪。
〖解说〗这是第三爻的爻辞，阳，表示主方态度坚定。
“沛”（pèi），充盛的样子：充沛。
“沬”（mèi），古同“昧”，微暗。
“肱”（gōng），胳膊由肘到肩的部分：肱骨。
主客双方激烈竞争，“丰其沛，日中见沬”。
主方的态度强硬，坚定地与客方抗衡，
如同日全食前，太阳留下牙形光亮区域，“折其右肱”。
主方的强硬态度是正确的，无所怪罪。

〖结构分析〗这条爻是阳爻在阳位，
当位，又与六阴有应。
当位表明主方的强硬态度有可能制约客方，
维护和扩大主方的利益，是潜在的对主方有利的因素；
有应表明客方态度随和，接受主方制约，
这潜在因素成了真正的对主方有利的因素。
第三爻是对第二爻的补充，
由于第二爻是阴爻，主方素质不佳，
主方的强硬态度缺少强力支持，
主方需要注意，不可强硬过分，
所以，爻辞只是说“无咎”，没有说“吉”。
```

### 四阳：丰其蔀，日中见斗，遇其夷主，吉。
```
它被盖住了很多，中午看到了北斗星，
遇到它夷平了主人，吉利。
〖解说〗这是第四爻的爻辞，阳，
表示客方积极主动地谋取和扩大自己的利益，比如说，
创新、创业、投资、进攻、求职、示爱，等等。
“夷”（yí），平，平坦，平安：化险为夷；
弄平：夷为平地；消灭：夷灭。
客方积极主动，在双方关系范围内，
与主方的积极主动全面冲突，
如同日全食的瞬间，太阳完全被月亮遮盖了，
天空不同于平常，“日中见斗，遇其夷主”。
由于主客双方的素质都不佳，主方态度强硬，客方态度随和，
这种情况已经到了冲突最严重的时候，
很快主方就会重新占优势，形势对主方有利。

〖结构分析〗这条爻是阳爻在阴位，
不当位，并且和一阳不有应。
不当位表明客方积极主动，
有可能损害主方利益，是潜在的对主方不利的因素；
不有应表明主方也是积极主动，
主方积极地维护和发展自己的利益，
这潜在因素没有成为真正的对主方不利的因素。
由于主卦的上爻是阳爻，
客卦的上爻是阴爻，主方的素质稍优于客方，
主方的力量稍强于客方，
主方有可能在竞争中获胜，所以爻辞说“吉”。
```

### 五阴：来章，有庆誉，吉。
```
美丽的时刻到来了，有美好的声誉，吉利。
〖解说〗这是第五爻的爻辞，阴，表示客方的素质不佳。
“章”（zhāng），在绘画或刺绣上，赤与白相间的花纹。
“庆誉”，美好的声誉。
在日食过程中，达到全食以后的瞬间，
太阳刚刚微露出一点点，一股极强的光环显现在太空，
像是一枚灿烂无比的巨大钻石戒指，“来章，有庆誉”。
就主客双方关系来说，客方的素质不佳，
这是主方发展的好机会，对于主方来说，“吉”。

〖结构分析〗这条爻是阴爻在阳位，
不当位，并且和二阴不有应。
不当位表明客方素质不佳，
有可能窃取主方利益，是潜在的对主方不利的因素；
不有应表明主方的素质也不佳，
并且主方积极主动，态度强硬，保护自己的利益，
这潜在因素没有成为真正的对主方不利的因素。
相反，由于主方的素质优于客方，
主方的力量强于客方，主方有可能利用客方的需要，
加强对客方的制约，
从而可以取得更多利益，所以爻辞说“吉”。
```

### 六阴：丰其屋，蔀其家，闚其户，阒其无人，三岁不觌，凶。
```
遮盖了他的房屋，让他的家盖上大草席，
从缝隙看他家，家中寂静无人，三年看不到人，有凶险。
〖解说〗这是第六爻的爻辞，阴，表示客方态度随和。
“闚”（kuī），同“窥”，从小孔、缝隙或隐蔽处偷看：窥探。
“阒”（qù），形容寂静：阒无一人。
“觌”（dí），相见：觌面（见面或当面）。
前面五条爻辞都是谈日食，到这条爻辞，
话题一下子从日食转到房屋上，表明，日食已经过去。
房屋被遮盖了，盖上了大草席，
这房屋被遗弃，所以“闚其户，阒其无人，三岁不觌，凶。
”因为日食是罕见的自然现象，这个“凶”可以理解为错过机会则凶。
从爻辞的字面意思理解，“其”指客方，
“闚其户，阒其无人，三岁不觌”，客方长期得不到发展。
客方态度随和，态度强硬的主方可能压制客方，使得客方得不到发展，
这么做，由于客方本来素质就不佳，
再得不到发展，对主方没有益处，有凶险。
主方不应当对态度随和的客方过于粗暴，过于傲慢，过于霸道。

〖结构分析〗这条爻是阴爻在阴位，当位，又与三阳有应。
当位表明客方态度随和，有可能接受主方制约，
是潜在的对主方有利的因素；
有应表明主方态度强硬，可以制约客方，
这潜在因素成了真正的对主方有利的因素。
尽管这种有利因素存在，
但是由于客方力量过于薄弱，可能对主方过分随和，
而主方的力量强于客方，
有可能态度强硬过度，这样不利于取得客方的信任，
不利于互动互利，结果不利于主方的发展，所以爻辞说“凶”。
这个判断词并不是说客方的随和态度对于主方有凶险，
而是说主方不能对客方强硬过度，不能失去客方的信任。
```

### 详细分析

《丰》卦是一个言及王师大举征伐之卦。也即仍是一个师旅之卦。它与言谈师旅的《师》、《旅》两卦不同的是：《师》卦乃言君王立师之道，《旅》卦乃是言将帅用兵之道，而《丰》卦则直接言谈君王帅兵亲征，《丰》卦卦辞中的“王假之”和“宜日中”就是这一意思。《丰》卦论及了王师在征伐过程中所遇到与荒淫无道之主作战的问题，遇到其邑国国君仍很精明国力仍很强盛应如何办的问题，遇到其邑国不甚黑暗仍有一些贤臣良将为辅弼当“用间”断除其左右股肱的问题，以及国人叛变夷除其君侯和敌国来章投降的问题，还有因杀戮过甚致使被征服的邑国“三岁不觌”的问题，可以说，这是一篇系统地论及王帅如何出征以及到灭亡各种类型邑国的专著。从另一个角度讲，《丰》卦所论及的各种情况绝不只是一种假设和虚构，应该说是一种记实。如“九四，丰其蔀，日中见斗；遇其夷主，吉”，这就和周文王帅大军征伐强大的密须（今甘肃灵台县）并迫使密须国人虏俘其国君投降就是一例。

```
丰①：亨。王假之②，勿忧，宜日中③。
初九，遇其配主④，虽旬无咎⑤；往有尚⑥。
六二，丰其蔀⑦，日中见斗⑧，往得疑疾⑨；
有孚发若⑩，吉。
九三，丰其沛⑪，日中见沬⑫；折其右肱⑬，吉。
九四，丰其蔀，日中见斗；遇其夷主⑭，吉。
六五，来章⑮，有庆誉，吉。
上六，丰其屋，蔀其家，窥其户，
阒其无人⑯，三岁不觌⑰，凶。

①丰：六十四卦卦名之一。
乃论述君王帅军亲自征伐其它邑国之卦。
②王假之：“假”（gé 格），至。
“王假之”，君王亲自前往。
③宜日中：宜日当午之时，喻当国势正盛之时。
④配主：“配”，此一字本为酒色之义（见《说文》），
后人借为妃字，失去其本义，变为匹配之义。
此处“配”仍为酒色原义。“配主”，
乃指贪恋酒色的荒淫无道之主。
⑤虽旬无咎：“旬”，
一月分为三旬，每十日为一旬。
此处“旬”，乃指多日。
此句为和其荒淫的配主作战，
就是荒废一些时日，也没有关系，也不会遭战败之祸。
⑥往有尚：指征伐无道昏君，会得到人们的拥护和崇尚。
⑦丰其蔀：“丰”，本丰富，此处当盛大的王师讲 。
“蔀”（bù 部），古谓日蔽云中，又有“大暗谓蔀”一说。
但按“上六”爻的“蔀其家”，
```

### 全卦内容

卦辞的“丰：亨。王假之，勿忧，宜日中”，是说作为象征王师大举进攻的《丰》卦，它亨通无比。
是说君王要帅大兵亲征，不要忧虑，只要在国力有如日当午的极昌盛时期出征就行了。

1. “初九”爻辞的“遇其配主，虽旬无咎；往有尚”，前半句是说君王征伐的对象是一个贪恋酒色、不务政务、不体察民情的荒淫之主，既作战的对象是这么一个国君，那就是作战荒废一些时日，也没有多大关系，也不会遭兵败之祸；后半句是说，王师应该继续征进，因为这是王师伐无道之举，会得到人们认可和崇尚的。
2. “六二”爻辞“丰其蔀，日中见斗，往得疑疾；有孚发若，吉”，前半句是说，当王师大兵压临到了一个国家时，却发现敌国的国君仍很精细，国力仍很强盛，不易攻克，这时不由不使征进军中的君王感到疑虑；后半句是说，当此之时，君王应该用自己诚实的信用去激励自己的将士和感化敌国的臣民，这样做是很吉祥的。这一爻颇有些承上一爻而来的意思。上一爻谈的是遇到昏君的办法，此一爻谈的是遇到敌国国君仍很精明国力仍很强盛时所应采取的办法。
3. “九三”爻辞的“丰其沛，日中见沬；折其右肱，无咎”，前半句是说当王师征进到一个尚不很腐败、尚不很黑暗的邑国，也即这个邑国仍有一些贤臣良将作为辅弼，不易攻克；后半句是说，当此之时，最好是用“用间” 的办法除掉这些股肱之臣，断掉其左右辅弼，这样做，可免除征进中的忧患。
4. “九四”爻辞的“丰其蔀，日中见斗；遇其夷主，吉”，前半句是说当王师大军压临到一个国君精明国力仍然强盛的国家时，由于部下将士的奋力战斗和对对方臣民的感化，迫使敌国臣民夷除其君侯前来投降，这种现象很吉祥。这一爻颇有些承“六二”爻而来。而且这一爻颇有些记实感，周文王帅大军征伐密须就是这样。密须当时很强盛，国君很精明，但在周文王的征进过程中，国君战败，后又被国人擒缚赴周文王军前投降。
5. “六五”爻辞的“来章，有庆誉，吉”，是说被征伐的邑国的君侯能识时务主动上表投降或遇其民变兵变夷除其君侯来上表投降，这对王师来说是值得庆贺和赞誉的，这样很吉祥。
6. “上六”爻辞的“丰其屋，蔀其家，窥其户，阒其无人，三岁不觌，凶”，此一爻的意思比较泛用，也是对上几爻的总结，是说不论遇到所侵伐的邑国的政权是彻底黑暗腐败还是不很黑暗腐败，但当战争结束获胜之后，王师所到之处，却发现其邑国或由于其原政权黑暗、人民流离失所，或由于战争杀戮过甚，致使户户无人，荒无人烟，大有三年不见一人之面之残迹，这样就很凶险。最后这一爻，也可以是对当时战争的总结，也可能是对战争的一种预见。不过几千年来的历史记载表明，自周以后的每次战争之后的数千里荒无人烟局面不能不说是这一爻辞的无数次上演。

这个卦是同卦，下震上震，相叠。震为雷，两震相叠，反响巨大，可消除沉闷之气，亨通畅达。平日应居安思危，怀恐惧心理，不敢有所怠慢，遇到突发事变，也能安然自若，谈笑如常。

### 九四：丰其蔀，日中见斗。遇其夷主，吉。《象》曰：丰其蔀，位不当也。日中见斗，幽不明也。遇其夷主，吉，行也。

将小席拼缀起来，躺下休息。正午时分，此人还在说看见北斗星，看来还未恢复正常。幸好遇着了他的老店主，把他托付给老店主，这一下可清静平安了。《象传》说：将小席拼缀起来，随地休息，是所处不得当，正如九四阳爻而处于阴位一样。正午时分看见北斗，也许天空迷暗不明的缘故。遇着他的老店主，这是吉利之行。

### 邵雍

平：得此爻者，明而受蔽，得人解释而吉。做官的受到领导和同僚的猜忌，位不安。

### 傅佩荣

- 时运：际遇不佳，由暗向明。
- 财运：走出暗昧，找到买主。
- 家宅：成败在人；巧遇良缘。
- 身体：眼疾待良医。

九四爻动变得[第36卦：地火明夷](e6988ee5a4b7mingyi_cn.md)。

这个卦是异卦，下离上坤，相叠。离为明，坤为顺；离为日；坤为地。

日没入地，光明受损，前途不明，环境困难，宜遵时养晦，坚守正道，外愚内慧，韬光养晦。

### 六五。来章，有庆誉，吉。《象》曰：六五之吉，有庆也。

赚得美玉，大家都庆贺夸奖他。这是吉利之兆。《象传》说：六五爻辞所讲的吉利，是因为有吉庆之事。

### 邵雍

吉：得此爻者，会得好人提举，谋望称意。读书人会取得佳绩。

### 傅佩荣

- 时运：实至名归，自然吉祥。
- 财运：经商有成，利名并至。
- 家宅：名门正户；天作之合。
- 身体：名医治之。

六五爻动变得[第49卦：泽火革](e99da9ge_cn.md)。

这个卦是异卦，下离上兑，相叠。离为火、兑为泽，泽内有水。水在上而下浇，火在下而上升。火旺水干，水大火熄。二者相生亦相克，必然出现变革。变革是宇宙的基本规律。

### 上六。丰其屋，蔀其家，窥其户，阒其无人，三岁不见，凶。《象》曰：丰其屋，天际翔也。窥其户，阒其无人，自藏也。

房子空荡荡的，屋顶上散乱盖着草席，从门缝里探视，寂无一人。看样子这里多年未住人了。这是不祥之兆。《象传》说：增修扩建房屋，看来此人如鸟飞蓝天，志得意满，发财不小。从门缝里探视，寂无一人，看来财多害身，横遭灾祸，他逃生去了。

### 邵雍

凶：得此爻者，骨肉相残，离祖成家，难免口舌之争。做官的位高者有危。

### 傅佩荣

- 时运：有命无人，不堪设想。
- 财运：无人经营，一筹莫展。
- 家宅：没落之家；婚姻不详。
- 身体：大限将至。

上六爻动变得[第30卦：离为火](e7a6bbli_cn.md)。

这个卦是同卦，下离上离，相叠。离者丽也，附着之意，一阴附丽，上下二阳，该卦象征火，内空外明。离为火、为明，太阳反复升落，运行不息，柔顺为心。
