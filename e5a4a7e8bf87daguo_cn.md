# 大过 ䷛

<img src="https://i.zgjm.net/d/file/20170331/325084ff2905ceb978c033a5c29c79a3.jpg" width=160 alt="大过" align=left>

大过（dà guò）卦是《周易》第28卦，代号是`3:6`，
主卦是巽卦，卦象是风、木，特征是顺，阳数是`3`；
客卦是兑卦，卦象是泽，特征是悦，阳数是`6`。
上泽下木，水漫过树，洪水之象，水多方酿成洪水。

“过”是过分的意思。“大过”，太过分。

卦形上本卦中间为四阳，上下为二阴，象征阳气过剩而失调。
当前形势对于主方来说是一个太过分的负担，主方不容易对付，但也不是灾难，只是主方需要谨慎处理。

易经六十四卦第28卦。 泽风大过（大过卦）非常行动。
简析，这个卦是异卦（下巽上兑）相叠。
阴阳爻相反，阳大阴小，行动非常，有过度形象，内刚外柔。

图中，红色表示当位的爻，天蓝色表示不当位的爻，箭头表示有应。

- 寓意：太过分
- 主卦：巽卦
- 客卦：兑卦
- 卦象：上泽下风（木）
- 卦序：28

## 哲学解读
### 起卦

大过是`3:6`卦的卦名，`3:6`是大过卦的代号。
过，是过分的意思。“大过”，太过分。卦名“大过”，说明当前形势对于主方来说是一个太过分的负担，主方不容易对付，但也不是灾难，只是主方需要谨慎处理。

`3:6`卦的主卦是`3`卦巽卦，卦象是风，特性是顺从；客卦是`6`卦兑卦，卦象是泽，特性是愉快。
主方只想顺从客方，而客方正享受成功的愉快，如何在与客方的关系中维护主方的利益，主方需要认真考虑。双方素质都良好，主方态度强硬，而客方态度随和，客方积极主动，而主方消极被动，主方不想有过分激烈的行为，而又不得不作些有效的活动以维护自己的利益。

起卦前需要客观地、全面地、准确地分析主方和客方情况，必须至少符合下列条件之一，此卦才有参考价值：

1. 主方与风很相似，客方与泽很相似。
2. 主方的阳数是`3`，客方的阳数是`6`。
3. 主方的行动是阴，素质是阳，态度也是阳；客方的行动、素质和态度是阳、阳和阴。

切勿用任何随机数方式起卦，否则此卦不能用作决策工具。

### 结构和卦爻辞

图左是大过卦的结构，其右是其卦爻辞，爻辞与爻，按由下而上的顺序，一一对应。
图中，有二个箭头，表示主卦和客卦的爻之间的有应关系，箭头方向是由阴爻到阳爻。
第一个箭头表示主方与客方的行动有应，主方消极被动，客方积极主动，处于和谐状态，
然而，这种和谐是主方受客方控制，不利于主方。
第一爻以天蓝色表示，不当位，这也表明主方消极被动不适当，主方应当设法扭转。
另一个箭头表示主方的态度和客方的态度处于和谐状态，客方接受主方要求，这对主方有利，
第三爻是红色，也表示主方态度正确，主方应当坚持强硬的态度。
没有应的一对爻是主卦和客卦的中爻，而且第二爻不当位，
不过，第二爻代表主方的素质，这条爻既不有应也不当位，不一定对主方不利，
这仅仅表示主方直接从客方素质受益的可能性小，而主方素质良好，则是主方实力所在，主方应当维护和加强。
总起来看，当前问题主要出在主方的消极被动上，主方应当极力扭转消极被动。

不过，主方状态是已经开始衰落的表现，已经经过了量变到质变的关键点，从乾卦衰落到巽卦，阳数减少了3，要返回到乾卦状态很难。
所以比较现实的办法是阴中有阳，在可能的情况下采取主动。

大过卦的卦辞中的“栋”象征主方，“栋挠”，栋梁超载弯曲，主方负担太重。爻辞形象地描述主方所处的状态，在超载过重的形势下，首先须要谨慎，“藉用白茅”；不要紧张，过载不一定是坏事，“枯杨生稊”；要警惕过载的严重程度，“栋桡”；要采取积极行动顶住压力，“栋隆”；避免作无意义的事，“枯杨生华”，要充分意识到过载的危险，“过涉灭顶”。删去少量判断词。卦爻辞就是一首围绕主题“大过”的散文诗。下面是其译文。

```
（尽管形势像）栋梁弯曲，还可以有所前进，
用白茅草铺垫，（谨慎小心）无所怪罪，
（就像是）枯杨树生芽，老翁得到少妻，没有不利。
栋梁弯曲了，有凶险，
（让）栋梁隆起，吉利；
（回避问题，）有其它关系，太不适宜，
（就像是）枯杨树开花，老妇得到少夫；
没有怪罪，没有赞誉。
（如果行动过度）涉过淹没头顶的深水，有凶险。
```

### 卦辞解释
```
〖卦辞原文〗栋桡，利有攸往，亨。

〖译文〗栋梁弯曲，利于有所前进，顺利。

〖解说〗消极被动的主方顺从客方，而客方积极主动，
使得主方感受巨大压力，“栋挠”，在客方压力下，
主方应当有所作为，进展会顺利，“利有攸往，亨”。
虽然利有攸往，并且亨通，然而栋梁弯曲，很难长久支撑。
```

### 初六
```
〖原文〗藉用白茅，无咎。

〖译文〗用白茅草铺垫，无所怪罪。

〖解说〗第一爻是主卦下爻，代表主方的行动，
阴，表示主方消极被动，比如说，防守、退却、不想冒险、
不想探索或开辟新的发展领域、只求保持现状，等等。
主方不想积极行动，在客方推动下，勉强行动起来，
不过很慎重，行动缓慢，行动之前作充分准备，“藉用白茅”，
如此，应当不受怪罪。
否则，不“藉用白茅”，行动仓促，则受怪罪。

〖结构分析〗第一爻的位置是阳位，这条爻是阴爻，
阴爻在阳位，不当位，不过与四阳有应。
不当位表明主方消极被动不适当，是对主方不利的潜在因素。
有应说明这种不利的潜在因素可能被客方利用，
对主方产生实际的不利效果，主方应当极力扭转消极被动局面。
但是，也不能仓促，爻辞提醒主方，“藉用白茅”。

【系辞上传】子曰：“苟错诸地而可矣；席用白茅，何咎之有？慎之至也。
夫茅之为物薄，而用可重也。慎斯术也以往，其无所失矣。”

译文：
孔子说：“随便地把祭品放置在地上就可以了，
却还要用白色茅草垫在下面作席子，还有什么灾难呢？
因为非常（祭祀的态度）慎重。
茅草这种东西非常微不足道，但是它的作用却非常重要。
把慎重地对待这些具体事情作为基础而继续发扬下去，
那就没有什么过失了。” 
```

### 九二
```
〖原文〗枯杨生稊，老夫得其女妻，无不利。

〖译文〗枯杨树生芽，老翁得到妻子，没有不利。

〖解说〗第二爻是主卦的中爻，代表主方素质，
阳，表示主方素质良好，比如说，
主方有资金、有地位、有权力、有实力、有技术，等等。
在客方推动下，基于主方的良好素质，主方意外地获益，
“枯杨生稊，老夫得其女妻”，这对于主方来说，没有不利的。
“枯杨”与“老夫”象征主方，“稊”与“女妻”象征客方。
有了“稊”可能生新枝，有了“女妻”可能生子女，
主方与客方的结合，为主方进一步发展带来了希望。
这一条爻辞是与上条联系的，主方先“藉用白茅”，
才有“枯杨生稊，老夫得其女妻”。

〖结构分析〗第二爻的位置是阴位，
这条爻是阳爻，阳爻在阴位，不当位，并且和五阳不有应。
不有应并不是对主方的不利因素，
只是说明客方从主方良好素质获益的可能性小。
这种不有应不仅对主方无害，
而且对主方有益，主方的良好素质被保存，
主方又可以用自己的良好素质获取利益，
爻辞形象地用“枯杨生稊，老夫得其女妻”作比喻。
```

### 九三
```
〖原文〗栋桡，凶。

〖译文〗栋梁弯曲，有凶险。

〖解说〗第三爻是主卦的上爻，代表主方态度，
阳，表示主方态度强硬。同时，第三爻是对第二爻的补充，
第二爻是阳，第三爻也是阳，这表示主方的素质很好。
主方也许很坚定，很明朗，也许很粗暴，很傲慢，
而客方态度随和，主方很难对付客方，形成“栋桡”状态，
如果主方不改过分强硬态度，栋梁可能折断，
所以说，“栋桡，凶”。
要避免凶险，就要改变主方过于强硬态度，
切不可粗暴，切不可傲慢。

〖结构分析〗第三爻的位置是阳位，这条爻是阳爻，
阳爻在阳位，当位，又与六阴有应。
当位表明主方的强硬态度是正确的，应当保持。
不过这种强硬态度是以良好素质为基础，
由于实力强大，可能自满、傲慢，或者粗暴。
而客方并不反对主方的强硬态度，反而顺从，
这可能又助长主方的傲慢和粗暴，其结果很危险。
```

### 九四
```
〖原文〗栋隆，吉；有它，吝。

〖译文〗栋梁隆起，吉利；有其它关系，太不适宜。

〖解说〗第四爻是客卦的下爻，代表客方行动，
阳，表示客方积极主动地谋取和扩大自己的利益，
比如说，创新、创业、投资、进攻、求职、示爱，等等。
“栋”象征主方，客方积极主动促进主方行动起来，
弯曲的栋梁窿了起来，主方开始行动起来了，这对主方是吉利的。
如果主方有其它关系，不顺从客方，主方就太不适宜了。

〖结构分析〗第四爻的位置是阴位，这条爻是阳爻，
阳爻在阴位，不当位，不过与一阴有应。
不当位表明主方可能受到积极主动的客方操纵，对主方不利；
有应表明客方确实操纵主方。形势对于主方来说并不好。
从另一方面说，主方消极被动本来就是主方的严重缺点，
如果，主方能够由此而振作起来，那就成了好事。
所以，主方不应当回避恶劣环境而找其它出路。
```

### 九五
```
〖原文〗枯杨生华，老妇得其士夫；无咎无誉。

〖译文〗枯杨树开花，
老妇人得到年轻丈夫；没有怪罪，没有赞誉。

〖解说〗第五爻是客卦中爻，代表客方素质，
阳，表示客方素质良好。在主客双方关系中，客方良好的素质，
给双方关系带来了光彩，好像是枯杨树开了花，只开花而不结子；
也好像老妇人得到了年轻丈夫，
有了年轻丈夫仍然不能生孩子，这些都没有实际效果。
主方已经有了良好素质，客方的良好素质对主方没有实际作用，
所以，对于主方来说，“无咎无誉”。

〖结构分析〗第五爻的位置是阳位，这条爻是阳爻，
阳爻在阳位，当位，然而与二阳不有应。
当位表明客方的良好素质是对主方有利的潜在因素，
主方有可能利用客方的良好素质获益；
不有应表明这种潜在的有利因素不成事实，
由于主方素质很好，比客方素质更好，比客方实力更强大，
主方不需要依靠客方的良好素质得益。
主方真正应当做的是着力扭转被动状态，
依靠自己的良好素质，开拓发展。
```

### 上六
```
〖原文〗过涉灭顶，凶，无咎。

〖译文〗涉过淹没头顶的深水，凶险，无所怪罪。

〖解说〗第六爻是客卦上爻，代表客方态度，
阴，表示客方态度随和。主方以强硬态度对待随和的客方，
可能陷入客方圈套，如同“过涉灭顶”，主方有凶险。
如果主方认识到这种凶险，改变过分强硬的态度，则无所怪罪。

〖结构分析〗第六爻的位置是阴位，这条爻是阴爻，
阴爻在阴位，当位，又与三阳有应。
既当位又有应，是对主方有利的因素。
本来，客方能够随和主方的强硬态度，
尊重主方意见，对于主方来说，这是好事，
主方可以让双方关系朝对自己有利的方向发展。
问题是，与第六爻对应的第三爻是对第二爻的补充，
第二爻和第三爻都是阳爻，表明主方实力非常强，
这就可能使得主方自满、傲慢和粗暴，
从而态度强硬过渡，“过涉灭顶，凶”。
主方应当以此为戒，态度强硬是正确的，
但是，切不可过分强硬。
```

### 大过释义
```
原文：
大过①：栋桡②。利有攸往，亨③。

译文：
大过卦：屋粱压得弯曲了。有所往则有利，通泰。

注释：
① 大过，卦名。本卦为异卦相叠（巽下兑上）。
上卦为兑，兑为泽；下卦为巽，巽为风为木。
上兑下巽，有泽水淹没木舟之象。
兑、巽相迭，中间四爻为阳爻，
初、上为阴爻，阳盛而阴柔，中壮而端弱，也兆示着拆毁之象。
喻人君人臣，行事大错，则将有栋拆粱摧之险。
所以卦名曰大过。过，过失。
② 栋，屋正中最高的横梁。
桡，沙少海先生说：
“原本作挠。《校勘记》曰：‘挠，各本作桡。’”
今从之。桡，弯曲。
③ “利有攸往，亨。”
此属另外一次贞卜的记录，文义上不过可强为贯通。
《易卦》作者将这凶吉绝然相反的占卜之辞同系于一卦，
意在反映大过的卦、爻之象预示着多种意义。
《彖辞》正缘此意进行解释。洋《彖辞》译文。
```

### 《彖》曰
```
原文：
《彖》曰：大过，大者过也。
“栋桡”，本末弱也。
刚过而中①，巽而说②行，
“利有攸往”，乃“亨”。
大过之时大矣哉。

译文：
《彖辞》说：大过，意思是大而过当。
“栋梁弯曲”，就是因为横梁中部太粗，两头太细，不堪负荷。
本卦阳爻多，阴爻少，阳刚过盛；
但是九二、九五之爻分居下卦、上卦之中位，
像人得贞正之道，有谦逊而和悦的品德。
秉此行事，“有所往则必有利”，所以又说“通泰”。
大过之卦，并容凶吉之象，因此其意义是重大的。

注释：
① 刚过而中。
本卦二、三、四、五，四爻为阳，
初、九为阴，所谓阳刚过盛。
“中”，以九二，九五之爻位为据。
② 巽而说，本卦上卦为兑，兑义为悦，下卦为巽，巽义为谦。

原文：
《象》曰：泽灭木，大过。
君子以独立不惧，遁世无闷。

译文：
《象辞》说：本卦上卦为兑为泽，下卦为巽为木，
上兑下巽，泽水淹没木舟，这是大过的卦象。
君子观此卦象，以舟重则覆为戒，
领悟到遭逢祸变，应守节不屈，稳居不仕，清静淡泊。
```

## 卦意
### 泽风大过
### 第28卦泽风大过（大过卦）非常行动
### 类型：中下卦

象曰：夜晚梦里梦金银，醒来仍不见一文，目下只宜求本分，思想络是空劳神。

这个卦是异卦（下巽上兑）相叠。
兑为泽、为悦，巽为木、为顺，泽水淹舟，遂成大错。
阴阳爻相反，阳大阴小，行动非常，有过度形象，内刚外柔。

### 新解
```
【原文】
（撰下兑上）大过①：栋桡②。利有攸往，亨。
初六：藉用白茅③，无咎。
九二：枯杨生梯④，老夫得其女妻。无不利。
九三：栋桡，凶。 九四：栋隆⑤，吉。有它⑥，吝。
九五：枯杨生华，老妇得其土夫。无咎无誉。
上六：过涉灭顶⑦，凶，无咎。

【注释】
① 大过是本卦的标题。大的意思是太，大过就是太过。
全卦的内容是讲 一些过了头的事，标题是按内容取的。
② 桡（nao）：弯曲。
③ 藉：席， 用作铺垫。白茅：一种柔软洁白，较贵重的草。
④ 梯：用作“荑”，意思 是草木新生、发芽。
⑤ 隆：中间高起来。
⑥ 它：指意外的事故。 
⑦ 灭顶：水淹过头顶。

【译文】
大过卦：屋梁被压弯了。有利于出门行旅，亨通。
初六：用白茅铺垫以示恭敬，没有灾祸。
九二：枯萎的杨树重新发芽，老头儿娶了年轻女子为妻。
没有什么不吉利。
九三：屋梁压弯了，凶险。
九四：屋梁隆起不弯，吉利。但有意外事故，不妙。
九五：枯萎的杨树重新开花，老妇人嫁了一个年轻丈夫。
没有灾祸也没有好处。
上六：渡河涉水，水淹过了头顶，凶险，但没有灾祸。

【读解】
对人对事采取中间态度，似乎是一种最好的选择：
过头或不及都失之偏颇。
相比之下，不及比过头还要好些，
所以才有“树大招风”、“高处不胜寒”这样的说法，
以及枪打出头鸟这样的做法。
中国人早已习惯了中庸的思维和行为方式，
这种内在心理结构，一时难以改变。
```
