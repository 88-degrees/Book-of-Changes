# 屯 ䷂

<img src="https://tenetai.com/88o/shapes/03.10.jpg" width=160 alt="屯" align=left>

屯（zhūn）卦，代号是`4:2`。
“屯”，囤聚。屯卦的主卦是震卦，客卦是坎卦 。

震卦的卦象是雷，“春雷一声惊万物”，震卦代表新生。
坎卦的卦象是水，水总是往下流。屯卦表示客方日益衰落的状态是主方兴起的良机，
同时客方力量仍然强大，主方的积极行动不一定取得好结果，主方应当耐心地囤聚力量。

屯卦，水雷屯，起始维艰。这个卦是异卦（下震上坎）相叠，震为雷，喻动；坎为雨，喻险。
雷雨交加，险象丛生，环境恶劣。“屯”原指植物萌生大地。
万物始生，充满艰难险阻，然而顺时应运，必欣欣向荣。
图中，红色表示当位的爻，天蓝色表示不当位的爻，箭头表示有应。

- 含义：起始维艰
- 代号：`4:2`
- 主卦：震卦
- 客卦：坎卦
- 卦序：3

### 屯卦简介

屯（zhūn)，屮（chè）字部，草木刚长出来的样子，一横象征土地，底下弯钩是象草根的弯曲，即象征草木初生，艰难的钻出土地，尚未伸展成形。《说文》：“屯，难也，象艸木之初生，屯然而难。从屮贯一。一，地也，尾曲。” 屯卦，《易经》中继乾、坤二卦后的第三卦，是由八卦中的坎、震二卦组合而成，内卦为震☳，一阳动于二阴之下，其德为能为动，其象为雷。外卦为坎☵，一阳陷于二阴之间，故其德为陷为险，其象为云为雨为水。震雷动能鼓动发育万物，坎水可滋养润化万物，春雷萌动万物始生，万物初生，屯然而难。内欲动而险在外，此卦意在突出事物初生时的艰难之象，然而顺应时运突破艰难的万物必欣欣向荣，故取名为“屯”。

### 总述

屯是`4:2`卦的卦名，`4:2`是屯卦的代号。

“屯”，囤聚。屯卦的主卦是震卦，客卦是坎卦。震卦的卦象是雷，“春雷一声惊万物”，震卦代表新生，代表新的起点，在八卦图中，震卦标志阳开始上升，另一方面，一阵雷鸣以后一般不造成什么实际损失，雷声大而短暂，实力薄弱，这也是新生事物幼嫩的特点。坎卦的卦象是水，水总是往下流，在八卦图中，从乾卦，经过巽卦到坎卦，坎卦表示事物处于衰落阶段，另一方面，水可以淹死人，水可以浮起大船，水可以冲走巨石，水可以泛滥成灾，水有巨大的力量。屯卦表示客方日益衰落的状态是主方兴起的良机，同时显示主方有很大的困难。客方力量仍然强大，主方的积极行动不一定取得好结果，主方应当耐心地囤聚力量。

起卦前需要客观地、全面地、准确地分析主方和客方情况，至少符合下列条件之一，此卦才有参考价值：
1. 主方与雷很相似，客方与水很相似。
2. 主卦阳数是`4`，客卦阳数是`2`。
3. 主方的行动是阳，素质是阴，态度也是阴；客方的行动、素质和态度是阴、阳和阴。

切勿用任何随机数方式起卦，否则此卦不能用作决策工具。

在屯卦结构与卦爻辞图中，您可以看到一条天蓝色爻，其余的爻都是红色的爻。
这条天蓝色爻给了我们什么信息呢？这是一条不当位爻，当位爻都是红色。
不当位的意思是这条阴爻在这个位置上不妥当。
为什么，因为这是主卦上爻，代表主方态度，如果主方态度像天一样地至高无上、不可冒犯，就有可能阻止客方损害主方利益，
因此这条爻应当是阳爻，所以，这条爻的位置是阳位。阴爻来了，占了阳的位置，所以说，不当位。
其结果是主方态度随和，有可能受到客方制约，从而使自己的利益受到损失，这是一个潜在的对主方不利因素。
是不是真地对主方不利，还得视客方的态度而定，如果客方态度强硬，主方确实受到客方制约，那么这潜在因素就成了真正的对主方不利因素。
然而，客方态度也随和，这潜在因素没有成为真正的对主方不利因素。

既然，主卦上爻是潜在的对主方不利因素，是否可以改变它呢，是不是主方立即对客方强硬起来就行了？
不是那么简单。主卦上爻是对主卦中爻的补充，主卦中爻和上爻都是阴，表明主方的素质非常差。
这随和态度是由极差的素质决定的，要改变对客方的态度，首先要增强自己的素质。
可怎么增强呢，客方比主方强，一旦客方发现主方企图增强素质，可能对客方不利，客方会打击主方的。
所以，主方需要悄悄地增强素质，装出一付对客方百依百顺的样子。这，恰恰是屯卦的卦爻辞所说的。

六条爻辞是一首完整的围绕主题“屯”的散文诗，下面是其译文。
```
像磐石和桓表坚定不移，坚持下去，
抓紧建立内部机构（力量薄弱，不要采取进攻性行动）。
有时驻守，有时难行不进，有时又乘马调动。
（让他相信，）不是盗匪，而是求婚者。
（他不相信）女子坚持不嫁，（以后就相信了）到十年就嫁。
追鹿没有准备，进入林中；您不如放弃，前进不恰当。
有时又乘马调动。（让他相信，）是来求婚。
（如果他相信）往前进就吉利，没有不利的事。
屯聚油脂（积蓄力量），短期坚持吉利，
长期坚持下去（他一旦发现真正意图）有凶险。
往复调动军马，流泪出血，痛哭不止。
（既要建军，又要让他相信是出于善意，
不得不忍受屈辱和刁难。）
```

当前形势是好的，但是主方的态度优柔寡断，过分随和，往前发展很难，总是徘徊不前，“屯如”，或“屯其膏”。
尽管主方的主观愿望很好，但是，不被客方理解。卦辞则指出在当前的好形势下，主方应当加强自身建设，下面逐条说明卦爻辞。

### 卦辞
```
〖卦辞原文〗屯：元亨，利贞，勿用有攸往。利建侯。

〖译文〗很顺利，利于坚持下去，不要往前走。利于建立侯国。

〖解说〗“元”（yuán）是第一。
“亨”（hēng），顺利。“贞”，坚持。
“利贞”指经中之“贞”皆训为“占”，解释为占问有利。
“攸”（yōu），助词，相当于“所”：性命攸关。
当前形势是很顺利的。应当坚持现有状态，不宜往前发展。
当前形势有利于健全内部组织。客方有可能帮助主方，
然而，主方须要努力取得客方信任，改善自己的素质。
屯为难，爻辞从初至上说的都有屯难的意思，
而卦辞却说“元亨利贞”，此卦和乾卦的初九爻意思相近，
“元亨利贞”是从卦的长远来说的。
中国有句老话说“万事开头难”，无论我们以去做什么，
创业之始必会遇到很多来自自身或外界的难题，
面对艰难险阻，只要我们坚定信念，耐的住辛苦，
勇于进取，敢于克服，自然风雨之后会有彩虹。
“勿用有攸往”，“勿”象形字，据甲骨文，像一旗。
右边是柄，左边是飘带。
本义：州里所建旗帜，古代大夫、士所建旗帜，
半赤半白，用来麾集警视人众。
这里引申为“不要，别”。
“用”象形，甲骨文字形，象桶形。
桶可用，故引申为用。这里引申为“可以施行”。
“攸”会意字，“行水也，从攴（pū），从人，水省 ”(《说文》)。
表示人扶杖走水路。本义：水流的样子。这里相当于“所”。
这句话的意思也就是说虽然坚信自己以后必会有所作为，
而在屯难之时却不要冒进妄动，要动心忍性，坚贞自持。
这就像在创业开始，虽然内心激情四射，但不要轻举妄动，
而要分析弄明各方面的情况，
审时度势，看准机会方可动，不然必会载个大跟头。
所谓“利建侯”：周易在当时是服务于天子的， 
所以这里的“建侯”应指天子建国封侯，
观此卦象，只有两爻为阳，九五处坎险之中，为坎险之主。
初九处震动之下，为震动主。
九五之君处险之中，而初九欲有所作为。
九五若起而伐之，必会招致灾难。
不如加以优待，封以侯伯，联合以共渡难关。故利建侯。
在一个创业团队中，每个人的性格和想法各有不同，
也不乏有些恃才孤傲，争功好利之人，
一个没有什么威望新的团队领袖若想驾驭每个人，
使团队团结有序，就要能识才用才，使每个人都能各安其位，
各司其职，各尽其能，这样便会无而不胜。
```

## 爻辞解释
### 初九
```
〖原文〗磐桓，利居贞，利建侯。

〖译文〗磐石桓表，利于安定坚持下去，利于建立侯国。

〖解说〗这是第一爻爻辞。
这条爻是阳爻，表示主方采取了积极行动，
积极主动地谋取和扩大自己的利益，
比如说，创新、创业、投资、进攻、求职、示爱，等等。
“磐”（pán），大石：磐石。
“桓“（huán）古代立在城郭、宫殿、官署、陵墓
或驿站路边的木柱：桓表。
这条爻辞的意思是，像坚石和桓表一样驻守原地，
有利于坚持安居，有利于健全内部建设。

〖结构分析〗第一爻位置是阳位，
这条爻是阳爻，阳爻在阳位，当位，并且与四阴有应。
当位，表明主方积极主动，
有可能利用客方的被动发展自己，这是潜在的对主方有利因素；
有应，表明客方的行动与主方的行动和谐，
主方可以利用客方消极被动发展自己，
这潜在因素成了真正的对主方有利因素，
主方应当雷厉风行地采取行动。
然而，由于主方素质极差，力量很弱，
主方不能急躁，不能盲目行动，因此，爻辞说不要盲动，“磐桓”。
主方应当首先从加强自身建设开始，所以，爻辞说“利居贞，利建侯”。
《象》曰：虽磐桓，志行正也。以贵下贱，大得民也。
```

### 六二
```
〖原文〗屯如，邅如，乘马班如。
匪寇，婚媾。女子贞不字，十年乃字。

〖译文〗有时驻守，有时不进，有时又乘马调动。
不是来掠夺的盗匪，而是来求婚。女子坚持不嫁，到十年就嫁。

〖解说〗这是第二爻爻辞。阴性，表示主方素质不佳，
比如说，资金缺乏、地位底下、实力薄弱、教育程度低，等等。
“邅”（zhān），难行不进。“班”，调回或调动（军队）：班师。
“字”，旧时称女子出嫁：待字闺中。
由于主方素质不佳，不能轻举妄动，
所以爻辞说，“屯如，邅如，乘马班如”。
主方需要向客方出示善意，“匪寇，婚媾”。
由于主方出示了善意，逐渐地客方会改变态度，
“女子贞不字，十年乃字”。

〖结构分析〗第二爻位置是阴位，这条爻是阴爻，
阴爻在阴位，当位，并且与第五爻有应。
当位又有应，是对主方的有利因素，
所以爻辞说“女子贞不字，十年乃字”。
《象》曰：六二之难，乘刚也。十年乃字，反常也。
```

### 六三
```
〖原文〗即鹿无虞，惟入于林中；君子几不如舍。往吝。

〖译文〗追鹿没有准备，进入林中；您不如放弃，往前太过分。

〖解说〗这是第三爻爻辞。阴性，表示主方的态度是阴，态度随和。
“虞”（yú），准备，防范。“惟”（wéi），文言助词。
“几”（jī），接近，达到。“吝”（lìn），过分爱惜：吝啬鬼。
这条爻的意思是：追鹿而没有准备就进入森林，
您去扑捉它不如舍弃它，前进是过分的行动。
爻辞以追鹿为比喻，说明主方应当对客方采取强硬态度，
但是还不具备条件，不能太性急。

〖结构分析〗第三爻位置是阳位，这条爻是阴爻，
阴爻在阳位，不当位，又与六阴不有应。
不当位，表明主方态度随和，
有可能受到客方制约，是潜在的对主方不利因素；
不当位，表明客方态度也随和，没有制约主方的意图，
那潜在因素没有成为对主方的真正不利因素。
如果主方改变态度，强硬起来，有可能反过来制约客方，
然而，主方素质太差，如果盲目前进，
可能很困难，所以，爻辞说“往吝”。
《象》曰：即鹿无虞，以从禽也。君子舍之，往吝，穷也。
```

### 六四
```
〖原文〗乘马班如，求婚媾；往吉，无不利。

〖译文〗乘马徘徊，求婚配；往前吉利，没有不利。

〖解说〗这是第四爻爻辞，阴，表示客方被动。
辞义是：往复调动军马去求婚，
往前进是吉利的，没有不利的事。
客方消极被动给主方的积极主动提供了机会，
主方应当抓住机遇，果断前进。

〖结构分析〗第四爻位置是阴位，这条爻是阴爻，
阴爻在阴位，当位，又和一阳有应。
当位，表明客方消极被动，
为主方发展提供了机会，这是潜在的对主方有利因素；
有应，表明主方积极主动，正好可以利用客方被动的机会，
那潜在因素成了真正的对主方有利因素。
所以，爻辞说“往吉，无不利”。
《象》曰：求而往，明也。
```

### 九五
```
〖原文〗屯其膏，小贞吉，大贞凶。

〖译文〗囤聚油脂，短期坚持吉利，长期坚持有凶险。

〖结构分析〗这是第五爻爻辞，
阳，表示客方素质良好。
“膏”（gāo）脂油：春雨如膏。
辞义是：充分利用客方的良好素质，
短暂保持现有关系对主方有好处，“小贞吉”。
因为主方是受惠于客方，这种情况不会维持很久，
主方应当着力于改善自己的素质，“大贞凶”。

〖结构分析〗第五爻位置是阳位，这条爻是阳爻，
阳爻在阳位，当位，又与二阴有应。
当位，表明客方素质良好，
有可能帮助主方，是潜在的对主方有利因素；
有应，表明主方素质不佳，需要客方帮助，
那潜在因素成了真正的对主方有利因素，
主方应当抓住这有利机遇，尽快地从客方的良好素质得益。
所以爻辞说，“小贞吉”。
这机会来自客方，如果主方行动迟缓，
可能失去良机，对主方不利，所以爻辞说“大贞凶”。
《象》曰：屯其膏，施未光也。
```

### 上六
```
〖原文〗乘马班如，泣血涟如。

〖译文〗乘马盘桓不进，哭泣不止。

〖解说〗这是第六爻爻辞。阴，表示客方态度随和。
“班如”（bān rú），盘桓不进貌。“涟”（lián）泪流不断的样子。
辞义是：往复调动军马，流泪出血，痛哭不止。

〖结构分析〗第六爻位置是阴位，
这条爻是阴爻，阴爻在阴位，当位，不过与三阴不有应。
当位，表示客方态度随和，有可能为主方的发展提供方便，
是潜在的对主方有利因素；不有应，表明主方态度也随和，
不能利用客方随和态度获益，
那潜在因素没有能够成为真正的对主方有利因素，错失良机。
所以爻辞说“泣血涟如”。

《象》曰：泣血涟如，何可长也。
```

### 屯卦注释
```
① 震：原始八卦之一。象征雷。
② 坎：原始八卦之一。象征水。
③ 屯：六十四卦卦名之一。乃始生之卦。
④ 利建侯：利于建立国君。
⑤ 磐桓“磐”（pán 盘），犹盘旋不进之义。
⑥ 邅如：“邅”（zhān 詹），犹豫绯徊之义。
⑦ 班如：亦犹盘旋不进之义。
⑧ 匪寇，婚媾：“匪”（fēi 菲），非的意思；“寇”，进犯之义。
“婚媾”，婚嫁迎娶。此句是说并非进犯，乃是婚嫁。
⑨ 字：许配之义。
⑩ 即鹿无虞：“即”，就的意思，此处指追逐。
“虞”，料想。此句是 说追逐鹿，没有料想到它跑掉了。
⑪ 君子几：朱熹注为“君子见几”，看来此处只能作此讲。
⑫ 吝：古指耻辱；也指恨惜。
《周易》中用法也不一。此处可能兼而有之。
⑬ 屯其膏：“膏”，指脂肪、油脂；
又指肥沃的土壤，如“膏腴”、“膏壤”。
此处泛指一切财力。“屯其膏”，指屯积财富和力量。
⑭ 小贞吉，大贞凶：指办小的正经事吉，办大的正经事凶。
⑮ 泣血涟如：泣血不止的样子。
```

### 屯卦详解
```
“初九”是屯卦的开始，意义重大。
阳爻位于阴爻的下方，以人事比拟，正是有才能，
又正当有利于建立公侯基业的草创时期，前途大有可为。

所以“象传”说：虽然踌躇，但志向、行为纯正，
只要不高高在上，能够与基层群众亲近，
就可以大得民心，获得拥戴。
这一爻，说明草创苦难的初期，虽然使人踌躇，
但也正是建功立业大有为的时期；正当即有利。

六二：屯如邅如，乘马班如。匪寇婚媾，女子贞不字，十年乃字。
《象》曰：六二之难，乘刚也。十年乃字，反常也。
“如”与若、然相同。“屯”是困顿、困难。“邅”进进退退。
“乘马”是四匹并列的马；“班”是行动不一致。
女子出嫁用簪将发束起，这时，于本名以外，
再起一字（号）以便于别人称得上，不必直呼姓名；
因而，女子出嫁说是“字”。
“六二”阴爻阴位，而且在下卦的中位，所以中正；
又与上卦的“九五”，阴阳“相应”，应当结为夫妻。
不料“六二”又恰好在阳刚的“初九”上方，
非常接近，以致“屯如”“邅如”，进退两难了。
就像并列的四匹乘马，脚步不一致，难以顺利前进。
《象传》说：“六二”的困难，是位于阳刚的“初九”的近上方。
十年才嫁，这是反常的现象。
这一爻的含义，说明在艰难的困境中，
必须意志坚定，不为威武所屈。不被反常的现象动摇。

六三：既鹿无虞，惟入于林中，君子几不如舍，往吝。
《象》曰：既鹿无虞，以纵禽也。君子舍之，往吝穷也。
“即”是就、近。“虞”是古代管理山林的官名。
“舍”同舍。“禽”同擒，当猎物解。
“吝”有惜、恨、耻的含义，比“悔”的程度高，接近“凶”。
“六三”阴爻阳位，因而不满，想要妄动。
但“六三”既不正，也不中，又与“上六”同是阴爻，
也不相应，轻率冒进，必然陷入困境。
以打猎追逐“鹿”来比拟。
如果没有管理山林的人向导，就不免会迷失在林中了。
因而，君子应当机警，不如舍弃，
如果冒然前往，就会有迷失被困在林中的耻辱了！
《象传》说：追鹿没有向导，是盲目的追逐猎物。
君子应当舍弃，前往会耻辱，因为将无路可走。
这一爻，说明应当知机，明辨取舍，不可盲目行动。

六四：乘马班如，求婚媾，无不利。
《象》曰：求而往，明也。
“六四”阴爻，本来与下卦的“初九”阴阳相应；
但却与上卦的“九五”过于接近，由于“初九”“九五”的相互牵制，
使“六四”意志动摇，犹如脚步不一致的四匹乘马，进退两难。
然而，“六四”毕竟与“九五”接近，只要向前去“求”，
就能够结合，所以说吉祥，没有不利。
《象传》说：向前追求，状况才能够明朗。
或者，向前追求，才是贤明的态度。
这一爻，说明当进退两难，抉择时困难，
应当采取积极的态度，惟有结合同志，向前追求，才能使状况明朗化。

九五：屯其膏，小贞吉，大贞凶。
《象》曰：屯其膏，施未光也。
“屯”是将难的意思当作动词用。
“膏”是油脂，转为恩泽的意思。
“九五”中正，又在最尊贵的“五”位；
然而，却陷在上卦“坎”的险陷的正中央，以致行动困难。
“九五”本来与“六二”阴阳相应；可是“六二”阴柔；
没有力量给以应援，不足以解困。
何况阳刚的“初九”，又在最基层受到拥护。
以致“九五”被困重阴中，孤立无援。
所以“屯其膏”，纵然有能力，也难以施展。
在这种状况下，如果是小事，保持纯正，还会吉祥；
如果是大事，即或保持纯正，也难免凶险。
《象传》说：在这种状况下，纵然要施展抱负，前途也未必光明。
这一爻，说明在孤立无援时，应当退守自保，不可逞强冒进。

上六：乘马班如，泣血涟如。
《象》曰：泣血涟如，何可长也。
“涟”是落泪。“上六”阴柔，
却上升到极点，已经是日暮途穷的时刻。
与下卦的“六三”，同属阴爻，不能获得应援，
以致陷于进无可取，退无可守的绝境；因而忧惧，血泪涟涟。
“象传”说：在这种状况下，又怎么能长久呢。
这一爻，以物极必反告诫。
在《易经》中，反复不断的以“满盈”提出警告。
```

### 全卦内容

此卦由雷下水上组成，其卦象有喻惊蛰春雷萌动万物始生之象。
卦辞“屯：元，亨，利，贞。勿用。有攸往，利建侯”，是说此卦虽有“元，亨，利，贞”之象，但由于是始生期，当如《乾》卦的初爻“潜龙”一样，切不可用。其后是说无论如何，但此卦毕竟还是“元，亨，利，贞”的，故有所往的，自然是利于去建立国君开拓国家的。

1. “初九”爻辞的“磐桓。利居正，利建侯”，是说此卦处在始生时期，所以其状将盘旋不前，起步艰难。但不论怎么讲，此卦前途无量，它仍利于居正，并利于建侯立国。
2. “六二”爻辞的“屯如邅如，乘马班如。匪寇，婚媾；女子贞不字，十年乃字”，前一句是说《屯》卦此卦有如犹豫不前之状，也有如乘着马在盘旋徘徊之中，难于前进。后一句是说此时虽并没有遇到什么敌方进攻，而还遇到有如喜庆的婚嫁迎娶；但是此时有如所遇到的女子忠贞坚守不嫁，待十年之后再嫁，还得一段时间，如果这样，暂时仍是可望而不可及。此爻乃言建侯立国仍得一段时间，仍得费一翻周折。
3. “六三”爻辞的“即鹿无虞，惟入于林中。君子几，不如舍；往吝”，是说在此《屯》卦之时，所追求的建侯立国一事，有如逐鹿，没有想到它逃脱跑入林中。当此之时，“君子”“大人”们见此之状，还不如暂时舍弃为好；如若再强行追逐，必然招致羞辱和悔恨。
4. “六四”爻辞的“乘马班如。求婚媾，往吉，无不利”，是说在此《屯》卦之时，其况总是有如“乘马班如”，盘旋不前。但是在此之时，还是要象求婚那样坚持下去，其结局自然是吉祥的，是顺利的。前一句是讲面临的现状；后一句讲的是前途与追求。
5. “九五”爻辞的“屯其膏。小贞吉，大贞凶”，前一句是讲当此《屯》卦之时，还是以屯积财富和积蓄力量为要。后一句是说在此之时，要去干一番建侯立国之事，还只能说是办一点小事则吉，办大事则凶。这里面有一个量力而行和超前负荷的问题。
6. “上六”爻辞的“乘马班如，其血涟如”，是说不论怎么讲，《屯》卦毕竟为始生之卦，尽管其前途无量，但它的起步仍是艰难的，它有如所乘的马盘旋不前，也有如痛苦于不进而泣血不止。此一爻可以说是对“君子”“大人”们处于《屯》卦时期现状的总结。

### 屯卦释文
```
屯：元、亨、利、贞。
虞翻曰：坎二之初，刚柔交震，故“元、亨”；之初得正，故“利、贞”矣。
勿用有攸往，利建侯。
虞翻曰：之外称往。初震得正，起之欲应，动而失位，故“勿用有攸往”。
震为侯，初刚难拔，故利以建侯。老子曰：善建者，不拔也。
《彖》曰：屯，刚柔始交而难生。
虞翻曰：乾刚坤柔，坎二交初，故“始交”。确乎难拔，故“难生”也。
崔觐曰：十二月阳始浸长，而交于阴，故曰“刚柔始交”。
万物萌芽，生于地中，有寒冰之难，故言“难生”。
于人事，则是运季业初之际也。动乎险中，大亨贞。
荀爽曰：物难在始生，此本坎卦也。

初六升二，九二降初，是刚柔始交也。
交则成震，震为动也，上有坎，是动乎险中也。
动则物通而得正。故曰“动乎险中，大亨贞”也。
雷雨之动满形，荀爽曰：雷震雨润，则万物满形而生也。
虞翻曰：震雷坎雨，坤为形也。谓三巳反正，成既济。
坎水流坤，故“满形”。谓雷动雨施，品物流形也。
天造草昧。荀爽曰：谓阳动在下，造物于冥昧之中也。
宜建侯而不宁。苟爽曰：天地初开，世尚屯难，震位承乾，故“宜建侯”。
动而遇险，故“不宁”也。
虞翻曰：造，造生也。草，草创物也。
坤冥为昧，故“天造草昧”。成既济定，故曰“不宁”，言宁也。
干宝曰：水运将终，木德将始，殷周际也。
百姓盈盈，匪君子不宁。
天下既遭屯险之难，后王宜荡之以雷雨之政，故封诸侯以宁之也。

《象》曰：云雷，屯。
《九家易》曰：雷雨者，兴养万物。
今言屯者，十二月雷伏藏地中，未得动出。
虽有云雨，非时长育，故言屯也。君子以经论。
荀爽曰：屯难之代，万事失正。经者，常驻也。论者，理也。
君子以经论，不失常道也。
姚信曰：经，纬也。
时在屯难，是天地经论之日，故君子法之，须经论艰难也。

《屯》卦，是《乾》《坤》两卦总论后的第一卦。
也可以说，《屯》卦实乃《乾》卦第一爻“潜龙勿用”的翻板。
这一点从《屯》卦辞的“元，亨，利，贞。勿用”可以得到证明。
历来都将此卦辞标点为“屯：元，亨，利，贞；勿用有攸往，利建侯”是不对的，
“勿用”与“有攸往”应该分开来标点，它们是两个各自独立的句子。
“勿用”有“潜龙勿用”的含义；
“有攸往，利建侯”是指《屯》卦的“元，亨，利，贞”这一哲理运用于政权本身。
从《屯》卦的卦辞“勿用”可以说明《屯》卦处于“潜龙勿用”的始生期，
而且从《屯》卦的各爻辞中也可以看到这一点。

“初九”爻辞为“磐桓”；
“六二”爻辞为“邅如”，“班如”，“女子贞不字，十年乃字”；
“六三”爻辞为“往吝”；“六四”爻辞中虽有“往吉”，但仍为“班如”；
“九五”爻辞为“小贞吉，大贞凶”；
“上六”爻辞为“班如，泣血涟如”，这些爻辞均为徘徊不前举步维艰之辞。

《屯》卦为《周易》起步的第一卦，为始生之卦，它讲的是一个事物始生起步之艰难。
由于《周易》一直将这些哲理运用于国家政权和王侯的崛起上，
所以，此开始一卦的卦辞和爻辞中便直言不讳地用了“利建侯”一辞。
```

### 总结

“屯”原指植物萌生大地。万物始生，充满艰难险阻，然而顺时应运，必欣欣向荣。屯卦是震之坎态，出生的艰难坎坷的状态。此卦的内涵是万物出生都难，但迟早会通过，利于坚持不懈，这时前往无用无为，但是有利于树立建功立业的雄心壮志。正确的态度是，既不轻举妄动，又利用不动的时机积极准备，自强不息。

新解：屯，念zhūn，平声。象形字。本义，小草初生的艰难。小草盖喻震，以其为阳元之始。坎者，险也。震动出而遇险，一象事业初创，险阻重重；一象事物初生，成长艰难。

象形字，本身就揭示为一颗小草其根从地下蜿蜒而上，直至冒出地表，长出两瓣小丫。屯，讲的是这个过程所具有的性质。同时，作者将屯卦本过程本具有的德性和卦爻发展的过程结合起来。这也是中国文字的精妙之处。所以，屯，又有囤积之意，此意可从小草生长的过程观象而得。

然而综合屯卦各爻辞，可以发现，实际上作者在这里，是将小草引申到小孩的成长以及事业的成长这两个方面。比如，从初爻的“磐桓，利居贞，利建侯。”初九至九五，互为山雷之颐，动而遇阻，出而后返，爻至六三出震体，故有“半远”而返象，另外艮为门，为家，故为止于家。读象为止于四，即言初爻象四岁内的孩子，父母不离身，是极少能独自出门的。故有“利居贞”，“利建侯”是延伸意，象事业初期打根基。古代王侯，分封而治。诸侯之间，当然象比邻之户。若根据传统所解释，“因为创办事业初期，应该好好打基础”，那么是不是就没有初期就“飞龙在天”，“如鱼得水”的呢？你我心中自知，不必多言。我们主张言而有根有据，不离象。虽易有独特个人阐发。但，中心的是客观的变化，是卦象本身具有的性质和变化，而不是自己的思想和经验的阐述。

往后几爻，大致讲小孩4 ～ 8岁，8 ～ 12，12 ～ 14、15，15 ～ 18、19，19 ～ 23、24岁。
分别从教育、生存技巧、婚姻、家庭。尤其最后一爻的“乘马烦如，泣血涟如”，实在是“少时不努力，老大徒伤悲”的真实写照，从哪里来？蒙卦之初爻是其源。同时，“泣血”也是大龄男女，难求婚姻的真情实感。这几爻爻辞可举一反三。

### 《文画易经》
```
《屯》：元亨，利贞。勿用有攸往，利建侯。
上六：乘马班如，泣血涟如。
九五：屯其膏，小贞吉，大贞凶。
六四：乘马班如，求婚媾，往吉，无不利。
六三：即鹿无虞，惟入于林中，君子几不如舍，往吝。
六二：屯如邅如，乘马班如，匪寇，婚媾。女子贞不字，十年乃字。
初九：磐桓。利居贞，利建侯。
《屯》：元亨，利贞。勿用有攸往，利建侯。
白话：初期的立足与成长：有元亨利贞四德，稍安勿躁，
踏踏实实的打好基础，这样才能逐步成为王侯。

初九：磐桓。利居贞，利建侯。
白话：要把自己置身于一个可激发志向的环境中（磐桓），
这样才能影响、逼迫自己寻求上进而不自甘堕落；

六二：屯如邅如，乘马班如，匪寇，婚媾。
女子贞不字，十年乃字。
白话：年轻人充满活力和热情（乘马班如，匪寇婚媾），
感觉有了热情可以克服一切困难获得成功，
但事实上任何人都不会喜欢没有自立能力的绣花枕头（女子贞不字）。

### 屯卦
```
屯：元亨，利贞。勿用有攸往，利建侯。
《彖》日：屯，刚柔始交而难生，动乎险中，大、亨、贞。
雷雨之动满盈，天造草昧，宜建侯而不宁。
《象》曰：云雷屯，君子以经纶。
有了天地，从而给万物有了生存的一席之地，故受之《屯》。
屯，屯立也。有屯立之所才能有发展之本，是以元始、亨通、吉利、中贞。
因此见勿用有攸往利建侯以存其德。

杂卦上说：“屯，见而不失其居也。”见，现也。
要展现自己就得不失其居，就得有支点，故再细小再微弱也要寻求一个立足之地。
因此屯的意思不外是立基定位的意思，草木无土则枯，
人无立业之地则危，有屯立之所是以元始、亨通、吉利、中贞。
因为只有有了屯立之所一切才能有所发展、繁衍、孕生、无处不在而生生不息。
故君子要想有所作为首先得有所割据才能立足于发展，
首先得寻找支撑点才能指望有所成功，否则一切就显得艰难和困苦。

《彖》辞说：屯，屯立也。原因是一切事物的形成与发展，
象刚柔始交时那样充满着艰辛和困苦，所有一切处处充满着险难，
因此只有屯立自我立足根本才能为发展而创造条件，
是以“可大，亨通，贞正”。其实，观乎大自然未尝不是处处充满着险难呢。
雷雨交加乌云密布充盈在天地之间，天下虽草创了万物又何尝宁静过，
所有一切无不体现着要立足发展只有立足根基面对严峻的挑战，
面对一切自然环境社会环境的挑战。
因此君王要有所建树宜于先建其侯，
立足其本，才能指望一切皆有发展和无限的延伸。

《象》辞说，乌云密布雷呜轰动处处险象环生这是《屯》之象，
君子观之思天下一切唯艰，从而当经纶天下，寻求可持续发展的道路。

初六：磐桓，利居贞，利建侯。
《象》曰：虽磐桓，志行正也。以贵下贱，大得民也。磐桓，盘桓也。
道路险难，故当先盘桓下来调整自我，是以“利居贞，利建侯”。
原因是草木有了盘根的场所才能生长发芽孕生发展，
动物有了隐藏之地才能栖息避害获得保护，
同样人类只有抛弃幻想寻找支撑点才能发展自我获得成功。
因此《象》辞说虽磐桓而不失行其志，这是发展之先决条件。
故君子当不耻下问，不以已贵，
以贵下贱而立足于民才能大得于民，这是万事之基也。

六二：屯如邅如，乘马班如，匪寇，婚媾。女子贞不字，十年乃字。
《象》曰：六二之难，乘刚也。十年乃字，反常也。
有屯立之所，才能有发展之本，因此当乘马而进，
不要怕会受到打击，而是会向更好方向发展。
因此“女子贞不字，十年乃字。”必须找合适机会。

其实，我们的一切努力都是为了向更好层次发展，
因此我们需要稳扎稳打创造更为有利的条件，而不是盲目挺进。
要嫁就嫁好君郎，不合时宜，宁待十年。
但十年乃字也太反常了，原因是时间和年龄我们容不得无限的等待和期望，
很多时候我们只有主动出击和容入社会中去努力创造条件。

六三：即鹿无虞，惟入于林中，君子几不如舍，往吝。
《象》曰：即鹿无虞，以纵禽也。君子舍之，往见吝也。
虞，防备的意思。无备之时措手不及，何以有成功。
可见，没有周密的计划就会受到意想不到的突变，
没有十足的把握就会遇到危险，因此君子不做无谓的努力，往见其吝也。
故君子即鹿无虞当纵其禽，待善而动方可无咎。

六四：乘马班如，求婚媾，往吉，无不利。
《象》曰：求而往，明也。
求婚媾，思合也。乘马班如，踏入而进也。
君子踏实而进，思合而化无有不吉不利也。
故求而往，思合而动，这是明智之举，有谁还会反对呢。
因为社会毕竟是一个共同体，容不得你想怎样就怎样。

九五：屯其膏，小贞吉大贞凶。
《象》曰：屯其膏，施未行也。
小富则安，小稳则喜，只能带来暂时的安稳，
因此君子当致富思源，富而思进，才能获得持续的发展。
浆泥在雉膏中，安乐于小富中，怎能应变大千世界？
故随遇而安没有进取心是没有光明前程的，其施不足以行天下也。

上六：乘马班如，泣血涟如。
《象》曰；泣血涟如，何可长也。
乘马班如，踏实而进也。泣血涟如，危难重重也。
这样的场面怎能长久存在呢？
穷则变，变则通，通则久，艰难的场面终将被人类所改善。
因为现实支配着人类必须不断的发展，
因为神奇的世界总是激发着人类无限的求知欲望，
因为活生生的生活摆在眼前，因为各种困难和疑惑促使着人类去发展和完善。
```

### 传统解卦
```
这个卦是异卦（下震上坎）相叠，震为雷，喻动；坎为雨，喻险。
雷雨交加，险象丛生，环境恶劣。

运势：身处困境，步步为营，有初难后解之意。

- 事业：起初多有不利，要知难而进，小心翼翼，勇往直前，灵活机动，可望获得大的成功，时机到来时一定要抓住，却也不得操之太急，且仍有困难，务必有他人相助，故平时应多施恩惠。
- 经商：创业初期步履艰难，多有挫折。坚定信念最重要，不要为表面现象所迷惑，应积极进取，行动果断，打开出路。若仍无法摆脱困境，则应退守保全，等待机会，再展宏图。

- 婚恋：好事多磨，忠贞纯洁，大胆追求，能够成功，婚姻美满。
- 决策：初始困难，但若具有坚忍不拔的毅力和锲而不舍的奋斗精神，前途不可估量，但往往不为他人理解而陷于孤独苦闷，事业会因此处于困难状态，需要得到贤德之人的帮助才能摆脱。如能以乐观主义精神处世，能取得成就。
