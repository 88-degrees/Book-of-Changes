# 巽 ䷸

<img src="https://i.zgjm.net/d/file/20170331/40f015506317a8c03dfbeaaec573a7f2.jpg" width=160 alt="巽" align=left>

巽是`3:3`卦的卦名，`3:3`是巽卦的代号。“巽”（xùn），古同“逊”，谦让恭顺。

`3:3`卦的主卦和客卦都是3卦巽卦，卦象是风，特性是顺从。
主方和客方都顺从，谁顺从谁呢？主方该如何行动呢？
主方应当保护好自己的利益，对客方谦让恭顺，能进则进，能退则退。图中，红色表示当位的爻，天蓝色表示不当位的爻。

巽卦是易经六十四卦第57卦，巽为风（巽卦）谦逊受益，象曰：一叶孤舟落沙滩，有篙无水进退难，时逢大雨江湖溢，不用费力任往返。

### 原文
```
巽下巽上。
巽①：小亨；利有攸往；利见大人。
初六，进退②，利武人之贞③。
九二，巽在床下④，用史、巫纷若⑤；吉，无咎。
九三，频巽⑥，吝。
六四，悔亡，田获三品⑦。
九五，贞吉，悔亡，无不利。无初有终⑧。
先庚三日，后庚三日⑨，吉。
上九，巽在床下，丧其资斧⑩，贞凶。

注释：
① 巽：六十四卦卦名之一。乃论述战前讨论和战后祝捷，
以及如何处理朝内私下对将帅的议论谗言之卦。
② 进退：即是进是退。此乃指战前商榷筹谋，
亦即《孙子兵法》所说的“庙算”，也犹今天的战前会议。
③ 利武人之贞：有利于将帅之贞正。
④ 巽在床下：此“巽”，犹耳边风，亦即意见和建议。
“床”，参《剥》卦“床”一辞，指王朝政权本身。
“巽在床下”，指一些朝臣非朝堂之上的私下进言。
⑤ 用史、巫纷若：“史”，用龟甲进行占卜之人。
“巫”，用蓍草进行求筮之人。此句是说借用卜筮向神灵求助。
⑥ 频巽：频频接纳众人意见，犹心无主见或朝令夕改。
⑦ 田获三品：政治术语，
犹“田有禽”、“田无禽”、“田获三狐”一类，此句是说战争收获甚著。
⑧ 无初有终：“无初”当指“九二”爻的“巽在床下”和“九三”爻的“频巽”。
“有终”即“六四”爻的“悔亡，田获三品”。
犹言开始很不顺利，但其结局却甚吉祥。
⑨ 先庚三日，后庚三日：此两句乃指战后祝捷的一种隆重仪式，
它与《尚书·周书·武成》在灭亡殷商王朝之后说的“执豆笾（biān 边），
越三日庚戌，柴望，大告武成”有相似处。
⑩ 丧其资斧：“资斧”一辞在《旅》卦中为兵器，
此卦中引申为执掌军旅之将帅。“丧其资斧”，指损坏我将帅辅臣。
```

### 词义
```
初六，进退，利武人之贞。
【白话】初六，过度谦卑，缺乏信心，
进退迟疑，利于勇武之人坚守中正之道。
《象》曰：“进退”志疑也；“利武人之贞”，志治也。
【白话】《象辞》说：“过度谦卑，缺乏信心，进退迟疑”，
是指意志懦弱犹豫；“利于勇武之人坚守中道”，
是勉励其修治，以树立坚强的意志。

九二，巽在床下，用史、巫纷若吉，无咎。
【白话】九二，过度谦卑而屈居于床下，
如果能像祝史、巫觋那样用崇敬谦恭的态度事神，
将十分吉祥，一定不会有什么祸患。
《象》曰：“纷若之吉”，得中也。
【白话】《象辞》说：“用崇敬谦恭的态度去行事将十分吉祥”，
这是因为九二爻能够居中守正的缘故。

九三，频巽，吝。
【白话】朝令夕改，使人无所适从，会有祸患。
《象》曰：“频巽之吝”，志穷也。
【白话】《象辞》说：“朝令夕改，使人无所适从，会有祸患”，
是因为当政者缺乏远大的志向。

六四，悔亡，田获三品。
【白话】六四，悔恨消失，田猎时得到多种收获。
《象》曰：“田获三品”，有功也。
【白话】《象辞》说：“田猎时得到多种收获”，
是因为能恪守“顺从”之道，所以才有所建树。

九五，贞吉，悔亡，无不利；
无初有终；先庚三日，后庚三日，吉。
【白话】九五，坚守中道，可以得到吉祥，
悔恨会消失，做任何事情没有不顺利的；
开始时也许不会太顺利，但最后一定会通达。
比如颁行新的法令、政令，
可以在象征变更的“庚”日的前三天发布，
在“庚”日后三天再开始施行这些命令，
才能使命令深入人心，从而使上下皆顺从，由此获得好的效果。
《象》曰：“九五之吉”，位中正也。
【白话】《象辞》说：
《巽卦》的第五爻位（九五）之所以吉祥，
是因为它居中端正，守持中道，慎始慎终。

上九，巽在床下，丧其资斧；贞凶。
【白话】上九，谦卑恭顺到了极点而屈于床下，
丧失了赖以谋生的资本，丧失了刚硬的本性，结果是凶险的。
《象》曰：“巽在床下”，上穷也；“丧其资斧”，正乎凶也。
【白话】《象辞》说：“谦卑恭顺到了极点而屈居于床下”，
处于穷极末路，无法前进；“丧失了谋生的资本”，
失去了生活的能力，结果必然是凶险的。
```

### 诠释内容

此卦由风下风上组成。卦辞的“巽：小亨；利有攸往；利见大人”，是说作为战前广泛征询意见，战后祝捷以及正确处理朝堂内部对将帅私下议论的《巽》卦，别小看它，它还是有小的亨通的；它亦利于前去办事；也利于显现君侯大人。

1. “初六”爻辞的“进退，利武人之贞”，是说在进行战争之前，广泛征询朝臣们的意见，讨论是进是退以及还是如何进如何退的问题，这样做，将有利于将帅们去进行作战。
2. “九二”爻辞的“巽在床下，用史，巫纷若；吉，无咎”，是说当朝堂中讨论作战的大计方针时，如果仍有一些人私下议论或幕后进言，意见不一，在这时最好用史巫进行卜筮借用神灵以明示；这样做将是吉祥的，也是可以免除灾祸的。实际上，这一种情况在殷周时期很普遍。在殷周时期，龟占与筮卜有时有决定权，只需龟筮认可，它不仅可以否决庶民的意见，否决朝臣的意见，而且也可以否决国君的意见，它有裁决权。
3. “九三”爻辞的“频巽，吝”，是说频频征求意见或频频接纳意见，心无主见，犹豫不决，朝令夕改，这样很不利，将会带来后悔和惋惜。
4. “六四”爻辞的“悔亡，田获三品”，是说能始终如一地贯彻执行经过“进退”而制定的政策，不要“频巽”，那后悔就会消亡，就能在战争中获得很大的胜利。
5. “九五”爻辞的“贞吉，悔亡，无不利。无初有终。先庚三日，后庚三日，吉”，此一爻实承上几爻的意见，第一句是说到此作战大获全胜之时，一切皆贞正吉祥，一切后悔皆尽消亡，一切一切皆无不顺利。第二句是说这一事一开始很不顺利（特别是指“巽在床下”），但是经过努力，最后却有一个“田获三品”的良好结局，真可谓是“无初有终”。最后一句是说在大获全胜、大功告成之后，应该举行一个“先庚三日，后庚三日”的盛大的庆捷活动，这样做，将是很吉祥的。
6. “上九”爻辞的“巽在床下，丧其资斧，贞凶”，这一爻可以说是一个回顾，一个自我反省与总结，是说当派遣将帅统兵在外作战时，千万不能听一些“床下”的私下议论和谗言而调遣回或残害一些栋梁的将帅之材，如果这样做，将是自毁我辅臣良将，这样做是很凶险。

### 介绍

这个卦是同卦（下巽上巽）相叠，巽（xùn）为风，两风相重，长风不绝，无孔不入，巽义为顺。谦逊的态度和行为可无往不利。

### 卦意

巽卦巽为风谦逊受益，中上卦，象曰：一叶孤舟落沙滩，有篙无水进退难，时逢大雨江湖溢，不用费力任往返。
这个卦是同卦（下巽上巽）相叠，巽为风，两风相重，长风不绝，无孔不入，犹如政令颁布，深入民心。巽义为顺。谦逊的态度和行为可无往不利。

### 结构和卦爻辞

巽卦的结构图中的爻有天蓝色和红色两种。天蓝色是不当位的爻，是潜在的对主方不利因素。
红色的爻是当位的爻，是潜在的对主方有利因素。主方应当利用潜在的有利因素，改变潜在的不利因素。不当位的爻有两条在主卦中，有一条在客卦中。主卦中的爻决定于主方，主方可以改变。客卦中的爻决定于客方，主方不能改变，然而可以观察，等待其变化。

主卦中的不当位的爻是第一爻和第二爻，代表主方的行动和素质。

第一条是阴爻，表示主方消极被动。由于消极被动，主方很难开拓进取，很难维护和扩张自己的利益，主方应当改变消极被动为积极主动。

第二条是阳爻，表示主方素质良好。在与客方交往中，有可能受到损失。主方应当低姿态，把良好素质隐藏起来，避免蒙受损失。

另一条不当位的爻是第六爻。这是客卦上爻，代表客方态度，阳，表示客方的态度强硬。而且，上爻是对中爻的补充，中爻和上爻都是阳，表示素质非常好，实力非常大。一方面，客方非常强大的实力给主方的扩张可能带来困难，另一方面，由于过分强大，有可能自满、傲慢和粗暴，客方的缺点暴露，有利于主方。主方应当耐心观察和等待，一旦客方缺点暴露，就可以积极主动地一举夺取最后胜利。

综合说来，主方应当转被动为主动，但不可盲动。应当隐藏自己的实力，冷静观察客方变化，能进则进，能退则退。

巽卦的爻辞详细说明了在主客双方实力相当情况下，主方应当采取的策略。主方应当保护好自己的利益，对客方谦让恭顺。谦让也不可完全被动，应当有进有退，“进退”；在谦让的时候，要把自己的利益保护好，隐藏起来，“巽在床下”；态度不要过于强硬，须要时时谦让，“频巽”；充分利用客方消极被动的机会，“悔亡”；坚持从客方的良好素质获益，“贞吉”，不要抢先制造事端，但要着手从事件的终了获益，“无初有终”；在遇到客方态度强硬的时候，要表现得谦让，把自己的利益隐藏起来，“巽在床下”。删去断辞，六条爻辞就是一首完整的围绕主题“巽”的散文诗，下面是其译文：

```
能进则进，能退则退，坚持刚武者的作法。
（有时）谦让地隐藏在床下，
（像）祝史和巫觋那样顺从自然。
频繁地谦让。（机会来了）在田野里获得多种猎物。
不要作事物的初始，但要作其终结；
先变更三天，再变更三天。
谦让恭顺（过渡了），隐藏在床下，
丢失了财货，继续下去有凶险。
```

卦辞则指出在当前形势下，主方应当有所作为。下面逐条说明卦爻辞。

### 卦辞
```
【卦辞原文】小亨，利有攸往，利见大人。

【译文】少许顺利，利于有所前进，利于表现为大人物。

【解说】主客双方的阳数比是3:3，
都不强，如果主方想有所作为，阻力不大，“小亨，利有攸往”。
主方应当趁客方消极被动之机，积极主动地发展自己，
不要对客方过于谦让，“利见大人”。
```

## 爻辞
### 一阴
```
【原文】进退，利武人之贞。

【译文】前进后退，利于坚持刚武者的作法。

【解说】第一爻是主卦下爻，代表主方的行动，
阴，表示主方消极被动，
比如说，防守、退却、不想冒险、不想探索或开辟新的发展领域、
只求保持现状，等等。同时，客方也是消极被动，
爻辞建议主方不要太消极被动了，应当趁客方消极被动之机，
或者前进，或者后退，像位刚武者，积极主动地行动。
“武人”，刚武的人。

【结构分析】第一爻的位置是阳位，这条爻是阴爻，
阴爻在阳位，不当位，并且和四阴不有应。
不当位，表明主方消极被动，可能不利于主方的发展，
具体如何，还有视客方的情况而定，这是潜在的对主方不利的因素。
不有应，表明客方也是消极被动，与主方不存在阴阳和谐关系。
由于阴阳趋向和谐是事物变化的动力，主方应当注意观察形势变化，
能进则进，能退则退，“进退”。
为了维护和发展自己的利益，
主方应当积极主动，像刚强的武人，“利武人之贞”。
```

### 二阳
```
【原文】巽在床下，用史巫纷若，吉，无咎。

【译文】谦让地隐藏在床下，用祝史和巫觋，
顺从天命的方式顺从，吉利，无所怪罪。

【解说】第二爻是主卦中爻，
代表主方素质，阳，表示主方素质良好，比如说，
有资金、有地位、有权力、有实力、有技术，等等。
由第五爻是阳爻可以知道，客方素质也是良好。
主方在与客方相互谦让恭顺的时候，
如何依靠自己良好素质获得更多利益呢？
须要把自己的良好素质隐藏起来，“巽在床下”，
像祝史和巫觋们顺从天命一般地顺从自然，
这样就吉利，无所怪罪。
“史巫”，祝史和巫觋；古代司祭祀﹑事鬼神的人。
“纷若”，盛多貌。“吉”和“无咎”都是以顺其自然为前提，
不顺其自然就不一定吉利，不一定无所怪罪。

【结构分析】第二爻位置是阴位，
这条爻是阳爻，阳爻在阴位，不当位，并且和五阳不有应。
不当位，表明主方的素质良好，有可能遭受损失，
主方应当低姿态，把良好的素质隐藏起来，“巽在床下”；
不有应，表明客方素质也良好，不一定需要损害主方利益，
那潜在因素没有成为真正的对主方不利因素，
如果主方能够顺从自然，“用史巫纷若”，
主方就可能保护甚至扩大自己的利益，“吉，无咎”。
```

### 三阳
```
【原文】频巽，吝。

【译文】频繁地顺从，有些不容易。

【解说】第三爻是主卦上爻，
代表主方态度，阳，表示主方态度强硬。
同时，客方态度也强硬，双方有冲突。
频繁有冲突，又要相互顺从，很不容易。
主方应当适当让自己的态度灵活些，柔和些。

【结构分析】第三爻位置是阳位，
这条爻是阳爻，阳爻在阳位，当位，然而与六阳不有应。
当位，表明主方的强硬态度有可能镇住客方，对主方有利，
这是潜在的对主方有利因素，具体是否有利，还有看客方的情况；
不有应，表明客方态度也强硬，
那潜在因素没有能够成为真正的对主方有利因素。
同时，第三爻和第二爻都是阳，
表示主方素质非常好，实力非常强大，
主方的态度有可能强硬过度。
主方态度须要灵活些，不要一直强硬，
而是应当时时谦让，“频巽”。这么做很不容易，“吝”。
```

### 四阴
```
【原文】悔亡，田获三品。

【译文】悔恨消亡，田野里获得多种猎物。

【解说】第四爻是客卦下爻，
代表客方的行动，阴，表示客方消极被动，等。
主方趁客方消极被动之机，发展自己的利益，“悔亡”。
“田”指双方关系，主方在双方关系中获得各种各样的利益，
“田获三品”。“三品”，上﹑中﹑下三等。

【结构分析】第四爻位置是阴位，
这条爻是阴爻，阴爻在阴位，当位，然而与一阴不有应。
当位，表明客方消极被动，有可能是主方发展的好机会，
这是潜在的对主方有利因素；不有应，表明主方也消极被动，
那潜在因素没有成为真正的对主方有利因素。
主方应当积极行动起来，这样就能够利用客方消极被动的良机，
没有悔恨，“悔亡”，从而获取利益，“田获三品”。
```

### 五阳
```
【原文】贞吉，悔亡，无不利；
无初有终；先庚三日，后庚三日，吉。

【译文】坚持下去吉利，悔恨消亡，没有不利的事；
不要作初始，要作终结；先变更三天，再变更三天，吉利。

【解说】第五爻是客卦中爻，
代表客方素质，阳，表示客方素质良好。
主方应当坚持双方关系，从双方关系中获益，
“贞吉，悔亡，无不利”。
不要主动开创事物的初始，消极被动地让客方去开创；
要积极主动地控制事物的进展，让事物在最有利于主方的时刻终结；
要不断地变更策略，每三天改变一次，吉利。
“庚”（gēng）变更，更换。

【结构分析】第五爻位置是阳位，这条爻是阳爻，
阳爻在阳位，当位，然而与二阳不有应。
当位，表明客方的良好素质有可能对主方有益，
这是潜在的对主方有利因素；
不有应，表明主方素质也良好，
没有从客方良好素质获益的兴趣，
那潜在因素没有成为真正的对主方有利因素。
形势是否有利主方，取决于主方的决策，
主方应当坚持变被动为主动，同时随和应对，
隐藏自己的实力，这样对主方吉利，“贞吉，悔亡，无不利”。
主方应当不要挑起事端，但要从事件牟利，“无初有终”。
冷静观察客方变化，灵活应对，
“先庚三日，后庚三日”，这样对主方吉利，“吉”。
```

### 定义

《巽》卦，也可以说是军旅之卦，或者说是前一《旅》卦的后续和补充。当然，它不像《师》卦《丰》卦《旅》卦那样去直接陈述战争，不能不说，《巽》卦也是一篇军事论著的重要组成部分。

姤（gòu）卦与夬卦相反，互为“综卦”。

姤即媾，阴阳相遇。但五阳一阴，不能长久相处。

## 九五爻详解
### 九五爻辞

九五。贞吉，悔亡，无不利。无初有终。先庚三日，后庚三日，吉。

象曰：九五之吉，位中正也。

### 白话文解释

九五：贞卜得吉兆，没有悔恨，无所不利。虽没有良好的开端，但有良好的结局。时日定在丁日或癸日，其事一定成功。

《象辞》说：九五爻辞之所以讲吉利，因为九五阳爻居上卦中位，像人事合于正道，自然吉利。

### 北宋易学家 邵雍 解

吉：得此爻者，谋望有成，无往不利。做官的先阻后顺。

### 台湾国学大儒 傅佩荣 解

- 时运：中正之位，无往不利。
- 财运：初有小悔，后得大利。
- 家宅：坐北朝南；相配得宜。
- 身体：三日可愈。

## 上九爻详解
### 上九爻辞

上九。巽在床下，丧其资斧，贞凶。

象曰：巽在床下，上穷也；丧其资斧，正乎凶也。

### 白话文解释

上九：人隐伏在床底下，钱财则被洗劫一空。卜问得凶兆。

《象辞》说：隐伏在床底下，正是上九阳爻穷途末路之象。钱财被洗劫，不正是凶险之事吗？

### 北宋易学家 邵雍 解

凶：得此爻者，多损失，或生疾病。做官的有运不逢时，须谨慎。

### 台湾国学大儒 傅佩荣 解

- 时运：越高越危，有失无得。
- 财运：因循失利，损耗不小。
- 家宅：有丧；惧内。
- 身体：或许告终。
