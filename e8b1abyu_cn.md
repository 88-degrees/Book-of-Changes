# 豫 ䷏

<img src="https://i.zgjm.net/d/file/20170330/fdbbe2c290fecb174bce1f2849f1539e.jpg" width=160 alt="豫" align=left>

豫卦是《易经》六十四卦之第16卦，代号是`0:4`，主卦是坤卦，卦象地，客卦为震卦，卦象雷。

雷地豫（豫卦）顺时依势。“豫”的意思是安闲，主方满足于现状，客方态度随和，为主方提供了安闲舒适的环境。

这个卦是异卦（下坤上震）相叠，坤为地，为顺；震为雷，为动。雷依时出，预示大地回春。因顺而动，和乐之源。

此卦与谦卦互为综卦，交互作用。《豫》卦是《谦》卦的对应卦，它从另一个侧面论述了建侯行师征邑国的策略问题。
这里面包括由于“鸣豫”带来泄密的恶果；包括不愿做豫备的 “盱豫”和迟迟不做豫备的“迟（豫）”带来的“有悔”。

- 卦义：由豫，大有得，恒不死。
- 主卦：坤卦（地）
- 客卦：震卦（雷）
- 卦序：16

### 简要介绍

豫卦阐述了干任何事情只有由于豫备才大有益处的“由豫，大有得”，以及由于豫备，一个国家才能永远不会灭亡的“恒不死”。
最后又阐述了由于某些豫备就绪之后，还可以改变与某些国家原来签署的盟约的“渝”盟，并申述了这种“渝”盟行为无罪无祸的“无咎”。

豫卦展示“豫”形势下各种变化的可能性。豫指安闲。
豫卦的代号是`0:4`，主卦是坤卦，卦象是地，阳数是`0`；客卦是震卦，卦象是雷，阳数是`4`。主方不要贪图安闲，应当努力改善素质。

### 全卦内容

此卦由地下雷上组成。
卦辞的“豫：利建侯行师”，是说《豫》卦乃论述豫备、豫防、豫谋之卦，它利于一个国家政权的建设，亦利于行师去征伐别的国家。

1. “初六”爻辞的“鸣豫，凶”，这是和《谦》卦“鸣谦，贞吉”对应的一爻，“谦”，可以“鸣”，“鸣”则吉祥，它可以利用他的声名去德服一些邑国；而“豫”则不可“鸣”，“鸣”则泄密，泄密则有覆军亡国之祸，故曰“鸣豫，凶”。
2. “六二”爻辞的“介于石，不终日，贞吉”，这一爻粗粗看起来好像与 《豫》卦无涉，但实际上则不然，也可以说它乃是设“豫”的前提，无它，“豫”则无法成立之。此句是说作为一个国君或将帅，能够在消息极为闭塞的情况下，能在不用一日便能了解情况，明白敌情，由于能及时了解情况，自然能早豫防之，如能这样，自然是贞正吉祥的。
3. “六三”爻辞的“盱豫悔；迟（豫）有悔”，是说对于豫备工作，忧愁害怕豫备不行，它会后悔；但明明知道有危险，仍迟迟不作豫备，将更后悔莫及。
4. “九四”爻辞的“由豫，大有得；勿疑，朋盍簪”，前半句是说，由于作了充分的豫备，一切已经就绪，自然就有所收益；后半句是说，既然一切豫备已经准备就绪，既然已经聚汇云集了天下所有的力量，实力已甚为雄厚，此时还有什么疑虑之有。
5. “六五”爻辞的“贞疾，恒不死”，是说一个国家若能常备不懈，既就 是其国家发生祸殃灾变，其国家也不会灭亡。
6. “上六”爻辞的“冥豫成，有渝无咎”，是说作为战争的敌对双方，为了需要，由于自己一方已暗暗地豫备就绪，就当改变原来的某种结盟性质或其它条文约束性的东西，这样将不至发生什么祸患。这一爻辞，实际上就是后来孙子所说的“兵以诈立”这一含义。这里仅指战争而言，它不牵涉背信弃义的道德问题。

### 结构和卦爻辞

豫卦下部三条爻是主卦，代表主方。主卦下爻和上爻不当位，以天蓝色表示，对主方不利。下爻是阴爻，表示主方消极被动，没有开拓进取精神，在与客方交往中，有可能遭受损失；上爻也是阴爻，表示主方态度随和，有可能受客方制约。至于中爻，这是当位的爻，以红色表示，表示主方素质不佳，这是主方谋求发展的动力，对主方有利。在主卦中可能产生的变化是下爻和上爻，由阴爻变为阳爻。这些变化决定于主方，主方可以改变自己的行动和态度，使得形势对自己更有利。

豫卦上部三条爻是客卦，代表客方。下爻与中爻不当位，对主方不利。下爻是阳爻，表示客方积极主动，正好可以利用主方的消极被动取得发展，有可能使主方遭受损失。中爻是阴爻，表示客方素质也不佳，客方不可能给主方实惠。主方不可改变这两条爻的状态，只有客方才能改变。从客方利益考虑，客方不需要改变这两条爻。上爻是阴爻，表示客方态度随和，对主方宽容，主方希望保持这条爻的状态，但是，客方希望改变它，变态度随和为态度强硬，这样可以利用主方的随和态度制约主方。因此，客卦中比较可能产生的变化是上爻的变化，从阴爻变为阳爻。

以上变化只是可能性，难以预测，是非常规变化。除此之外，还需要考虑常规变化。

常规变化是事物自身的发展变化，循螺旋线规律，用阳数表示，变化的一圈是4、5、6、7、3、2、1和0。
主卦的阳数是`0`，是一圈的终点，主方目前的状态是长期衰落的结果。由于常规变化规律的存在，主方要完全扭转消极被动很困难，只能是阴中有阳，在一定范围内采取主动。主方的随和态度是素质极差的表示，也不可能有明显变化。

客卦的阳数是`4`，是常规变化一圈的起点，具有很强生命力，但是实力很弱，态度随和是素质不佳的表现。不佳的素质和随和的态度留给了主方一个安闲环境。主方应当尽量维持当前状态，享受安闲。

客卦的阳数从4增加到5，是一个量变，发生的可能性很大。结果，客卦从震卦变成离卦，上爻从阴爻变成阳爻，那时，主方将受到客方制约，主方的处境更差。主方需要有此准备，不断改善自己的素质，为从坤卦跃变到震卦作准备。

图的底部是卦名和卦辞，卦辞从总体上说明主方应当加强自身建设。

图的右边是爻辞，从下到上，与爻一一对应，形象具体地说明爻的含义。
第一爻、第三爻、第四爻与第六爻都提到了“豫”，分析了安闲环境下的各种情况。
首先，安闲的时候，不应当一边说，一边贪图安闲；
其次，应当为自己贪图安闲而后悔，后悔得越早越好；应当跟随别人行动起来；
贪图安闲是愚昧的，应当改变贪图安闲的状态。

删去判断词，可以看出，六条爻辞是一首围绕主题“豫”的散文诗。下面是其译文：

```
安闲，只说不做，
在光秃的岩石之间，挨不到日终。
安闲，回顾往事有悔，行动迟缓更有悔。
安闲，由别人作主，大有得益，
不要怀疑，朋友怎不给插上发簪。
有疾病长期坚持，就是总死不了。
安闲，成为黑暗愚昧，(应当)有变。
```

### 卦辞解释
```
豫：利建侯、行师。
郑玄曰：坤，顺也。震，动也。
顺其性而动者，莫不得，得其所，故谓之豫。
豫，喜逸说乐之貌也。震又为雷，诸侯之象。
坤又为众，师役之象。故“利建侯、行师”矣。

虞翻曰：复初之四，与小畜旁通。
坤为邦国，震为诸侯。
初至五，体比象。
四利复初，故“利建侯”。
三至上，体师象，故“行师”。
《彖》曰：豫，刚应而志行，
侯果曰：四为卦主，五阴应之，
刚志大行，故曰“刚应而志行”。
顺以动，豫。
崔觐曰：坤下震上，顺以动也。
豫顺以动，故天地如之，

虞翻曰：小畜，乾为天。坤为地。
如之者，谓天地亦动，以成四时，而况建侯行师。
言其皆应而豫也。而况建侯行师乎！
《九家易》曰：震为建侯，坤为行师；
建侯所以兴利，行师所以除害。
利兴害除，民所豫乐也。天地有生杀，万物有始终。
王者盛衰，亦有迭更，犹武王承乱，而应天地。
建侯行师，奉辞除害，民得豫说，君得安乐也。
天地以顺动，

虞翻曰：豫变通小畜。坤为地。
动初至三成乾，故天地以顺动也。
故日月不过，而四时不忒。

虞翻曰：过谓失度，忒，差迭也。
谓变初至需，离为日，坎为月，
皆得其正，故“日月不过”。
动初时，震为春。至四，兑为秋。
至五，坎为冬。离为夏。
四时位正，故“四时不忒”。通变之谓事，盖此之类。
圣人以顺动，则刑罚清而民服。

虞翻曰：清，犹明也。动初至四，兑为刑。
至坎，为罚。坎、兑体正，故“刑罚清”。
坤为民，乾为清，以乾乘坤，故“民服”。
案：帝出震，圣人也。坎为法律，刑罚也。
坤为众，顺而民服也。豫之时义大矣哉。

虞翻曰：顺动天地，使日月四时，皆不过差。
刑罚清而民服，故“义大”也。
《象》曰：雷出地奋，豫。
崔觐曰：震在坤坤，故言“雷出地”。
雷，阳气，亦谓龙也。
夏至后，阳气极而一阴生。
阴阳相击，而成雷声。
雷声之疾，有龙奋迅豫跃之象，故曰“奋豫”。
先王以作乐崇德，殷荐之上帝，以配祖考。
郑玄曰：奋，动也。
雷动于地上，而万物乃豫也。
以者，取其喜佚动摇，犹人至乐，
则手欲鼓之，足欲舞之也。
崇，充也。殷，盛也。荐，进也。
上帝，天帝也。
王者功成作乐，以文得之者，作钥舞；
以武得之者，作万舞；各充其德而为制。
祀天帝以配祖考者，使与天同飨其功也。
故《孝经》云：郊祀后稷以配天，
宗祀文王于明堂，以配上帝是也。

综述：（雷地豫卦）

上震下坤，中存坎艮，上方动验，中方满险，
下又止之，则其险无所用矣，雷在地上，震惊万物，
屈者伸，藏者露，顺以动之，动以顺豫，
君子则为逸豫之象，有贵合五行吉。
豫者悦也，卦中缺父母，以震卦初爻子水父母，
伏于本卦初爻未土之下，未土是飞神，
子水是伏神，土克水，谓之飞来克伏。
```

### 爻辞解释
```
初六：鸣豫，凶。
象曰：初六鸣豫，志穷凶也。
虞翻曰：应震善鸣，失位，故“鸣豫，凶”也。
虞翻曰：体剥蔑贞，故“志穷凶也”。

六二：介于石，不终日，贞吉。
虞翻曰：介，纤也。
与四为艮，艮为石，故“介于石”。
虞翻曰：与小畜通，应在五。
终变成离，离为日，得位。
欲四争复初，已得休之。故“不终日，贞吉”。
侯果曰：得位居中，柔顺正一，
明豫动之可否，辩趣舍之权宜。
假如坚石，不可移变，应时则改，不待终日。
故曰豫之正吉。
象曰：不终日，贞吉；以中正也。

六三：盱豫，悔。 迟有悔。
王弼曰：履非其位，承动豫之主。
若其雎盱而豫，悔亦至焉。
迟而不从，豫之所疾，进退离悔，位不当也。
向秀曰：雎盱，小人喜说、佞媚之貌也。
象曰：盱豫有悔，位不当也。

九四：由豫，大有得。勿疑。朋盍簪。
侯果曰：为豫之主，众阴所宗，莫不由之以得其豫。
体刚心直，志不怀疑，故得群物依归，
朋从大合，若以簪菾之固括也。
虞翻曰：由，自从也。
据有五阴，坤以众顺，故“大有得”。
得群阴也。坎为疑，故“勿疑”。小畜兑为朋。
盍，合也，坤为盍。戠，聚会也。
坎为聚，坤为众，众阴并应，故“朋盍戠”。
戠，旧读作撍，象曰：由豫，大有得；志大行也。

六五：贞疾，恒不死。
虞翻曰：恒，常也。坎为疾。
应在坤，坤为死。震为反生。
位在震中，与坤体绝。故“贞疾，恒不死”也。
象曰：六五贞疾，乘刚也。恒不死，中未亡也。

上六：冥豫，成有渝，无咎。
虞翻曰：应在三。坤为冥。渝，变也。
三失位，无应，多凶。变乃得正，体艮成。
故“成有渝，无咎”。

象曰：冥豫在上，何可长也。

坤下震上。
豫①：利建侯行师。
初六，鸣豫②，凶。
六二，介于石，不终日③，贞吉。
六三，盱豫悔④；迟有悔⑤。
九四，由豫⑥，大有得；勿疑，朋盍簪⑦。
六五，贞疾，恒不死⑧。
上六，冥豫⑨成，有渝⑩无咎。

注释：
① 豫：六十四卦卦名之一。
乃论述为建侯行师所设豫备、豫防、豫谋之卦。
② 鸣豫：与《谦》卦的“鸣谦”相对应，
指大张旗鼓地宣扬豫备、豫防、豫谋等，含有泄密之义。
③ 介于石，不终日：“介于石”，
有《困》卦的“困于石，
据于疾藜” 之义，指处境困难，消息闭塞。
“不终日”，指不到一日便明白掌握了情况。
全句是说如能在消息极为闭塞的情况下，
也能在不到一天的时间内掌握敌情。
④ 盱豫：“盱”（xū 需），忧愁。
“盱豫”，指害怕和不愿意做豫备工作。
⑤ 迟有悔：此句全句当“迟豫有悔”。
即迟迟不作豫备，后悔无穷。
⑥ 由豫：“由”，由于。“由豫”，由于豫备。
⑦ 朋盍簪：“朋”，群（见《泰》卦注⑧）。
“盍”，合。“簪”，为古“（无左上+竖）”的俗字，
首笄，即今束发的簪子。
此处“朋”“盍”“簪”皆有聚汇云集之义。
“朋盍簪”，实乃《泰》卦“拨茅茹，以其汇”的另一种用法。
⑧ 贞疾，恒不死：“疾”，灾祸，危难。
“恒不死”，永不会灭亡。
全句是说一个国家能常备不懈，
即就是出现一些灾祸和危难也不会灭亡。
⑨ 冥豫：“冥”，暗。“冥豫”，暗暗豫备。
⑩ 有渝：“渝”，改变。“有渝”，有所改变。
此处指改变原来某种结盟性质或缔约性质的东西。
```

### 卦辞概述
```
〖卦辞原文〗利建侯行师。

〖译文〗利于建立诸侯国和出兵打仗。

〖解说〗这是豫卦卦辞。
“利建侯行师”的字面意思是，
“利于建立诸侯国和出兵打仗”。
“建侯行师”是个用例，或者比喻。
“建侯”指主方自身的建设。
“行师”指主方对客方以及外界的交往。
“利于”的事情是适合做的事情，
在这里“利于”可以理解为“应当”。
就是说，“利建侯行师”的意思是，
主方应当从自身建设和对外交往二个方面积极行动。
这五个字的卦辞，突出建议主方的行动方向，
不是说整个形势对主方有利，
而是说在当前安闲舒适的状态下，主方应当积极行动起来。
对于主方来说，“建侯”可以理解为增强自己的素质，
例如学习技术，丰富经验，增强工作能力等等。
“行师”指出兵打仗，这也是比喻。
对于主方来说，“出师”可以理解为主动与客方联系，
增进相互了解。综合起来，“利建侯行师”意思是：
主方应当从自身建设和对外交往二个方面积极行动。
这样，才有可能维持安闲舒适的环境。
```

### 一阴
```
〖原文〗鸣豫，凶。

〖译文〗鸣叫而安闲，有凶险。

〖解说〗这是豫卦第一爻爻辞。
第一爻表示主方的行动。
这是一条阴爻，表示主方消极被动，
比如说，防守、退却、不想冒险、
不想探索或开辟新的发展领域、只求保持现状，等等。
“鸣”是发出声音，鸟兽叫为鸣，
“豫”是安闲舒适的意思，“凶”是不幸。
这条爻辞是对主方的警告：
只说而不行动，满足于安闲舒适，是不幸的。

〖结构分析〗第一爻是主卦下爻，
其位置是阳位，这条爻是阴爻，
阴爻在阳位，不当位，然而与四阳有应。
不当位，表明主方消极被动，
有可能被客方攻击遭受损失，是潜在的对主方不利因素；
有应，表明客方积极主动，
正好可以利用主方消极被动的机会取得发展，
这潜在因素成了真正的对主方不利因素，所以爻辞说“凶”。
```

### 二阴
```
〖原文〗介于石，不终日，贞吉。

〖译文〗在石头中间，
难于维持到一日的终了，坚持下去吉利。

〖解说〗这是豫卦第二爻爻辞，
阴，表示客方素质不佳，比如说，
资金缺乏、地位底下、实力薄弱、教育程度低，等等。
“介于石”形容主方的安闲状态，
在石头中间，狂风巨浪冲不走，野兽侵害不着。
“不终日”形容安闲的状态很难维持，
在石头中间，无饮无食，维持不了一天。
“贞吉”中的贞是坚持的意思，
在这里，是指坚持主客双方关系的现有状态，
而不是说主方应当坚持消极被动。
主方应当改变消极被动状态，如果主方改变消极被动，
响应客方的积极行动，对主方说来是吉利的。
综合起来，这条爻辞的意思是，
客方提供了安全环境，使主方能够安闲，像介于石头中间。
但是，安闲状态不能长久，不终日。
主方应当响应客方的积极行动，维持与客方的关系，
如果这样，是吉利的。

〖结构分析〗第二爻是主卦中爻，其位置是阴位。
这条爻是阴爻，阴爻在阴位，当位，然而与五阴不有应。
当位，表明主方素质不佳，是积极寻求发展的动力，
是潜在的对主方有利因素；
不有应，表明客方素质也不佳，
主方不可能从客方获得利益，
这潜在因素没有成为真正的对主方有利因素。
由于这种潜在因素的存在，如果主方坚持下去，
也可能发现得利的机会，所以爻辞说“贞吉”。
```

### 三阴
```
〖原文〗盱豫悔，迟有悔。

〖译文〗回顾安闲悔恨，迟缓有悔恨。

〖解说〗这是豫卦第三爻爻辞，
第三爻代表主方态度，阴，表示主方态度随和。
“盱豫悔”中的“盱”是意思是“睁开眼向上看”。
如果理解为“回顾”，
则“盱豫悔”的意思是“回顾安闲的生活，有所悔恨”。
“迟有悔”中的“迟”指行动迟缓，
这句话的意思是“行动迟缓将有悔恨”。
客方在积极行动，主方应当意识到自己的素质太差，
应当迅速响应客方而积极行动起来。
如果主方跟不上步伐，可能会有悔恨。
这条爻辞中，二个“悔”字的意思不一样，
前一个“悔”指过去和现在的悔恨，
后一个“悔”指未来可能有的悔恨。

〖结构分析〗第三爻是主卦上爻，
其位置是阳位，这条爻是阴爻，
阴爻在阳位，不当位，和六阴不有应。
不当位，表明主方态度随和，
有可能受客方制约，是潜在的对主方不利因素；
不有应，表明客方态度也随和，客方不求制约主方，
这潜在因素没有成为真正的对主方不利因素。
所以爻辞中没有判断词。
```

### 四阳
```
〖原文〗由豫，大有得；勿疑，朋盍簪。

〖译文〗顺从安闲，很有收益；
不用怀疑，朋友何不插戴。

〖解说〗这是豫卦第四爻爻辞，
代表客方的行动，阳，
表示客方积极主动地谋取和扩大自己的利益，比如说，
创新、创业、投资、进攻、求职、示爱，等等。
“由豫，大有得”中的“由”是顺随，听从的意思。
客方的行动是阳，主方的行动是阴，主方听从客方。
这五个字的意思是：主方听从客方，安闲舒适，大有所得。
“勿疑，朋盍簪”中的“朋”指朋友，这里指客方，
“盍”是疑问副词，相当于“何不”的意思。
“簪”是（在头上）插戴的意思。
这五个字的意思是：不用怀疑，
客方何不在主方的头上插戴（一朵花）？
综合起来，这条爻辞的意思是“主方听从客方，
安闲舒适，大有所得；
不用怀疑，客方何不在主方头上插戴（一朵花）”。
前五个字建议主方听从客方，从现在被动的安闲舒适状态，
积极行动起来，这样做会大有所得。
后五个字是警告主方，客方肯定会促使主方积极行动起来。
自觉或不自觉，主方都必须顺从客方，积极行动起来。

〖结构分析〗第四爻是客卦上爻，
其位置是阴位，这条爻是阳爻，
阳爻在阴位，不当位，然而与一阴有应。
不当位，表明客方积极主动有可能损害主方利益，
是潜在的对主方不利因素；
有应，表明主方消极被动是客方发展的好机会，
这潜在因素成了真正的对主方不利因素。
```

- 时运：大运正行，不必担心。
- 财运：众货聚集，大有利市。
- 家宅：得福有财。
- 身体：不必担心。

### 九四变卦：雷地豫 变卦 坤为地

<img src="http://www.yilusoso.com/wp-content/uploads/2018/06/16_4.png" align=left>

九四爻动变得[第2卦：坤为地](./e59da4kun_cn.md)。

这个卦是同卦（下坤上坤）相叠，阴性。

象征地（与乾卦相反），顺从天，承载万物，伸展无穷无尽。

坤卦以雌马为象征，表明地道生育抚养万物，而又依天顺时，性情温顺。

它以“先迷后得”证明“坤”顺从“乾”，依随“乾”，才能把握正确方向，遵循正道，获取吉利。

## 六五爻详解
### 六五爻辞

六五。贞疾，恒不死。

象曰：六五贞疾，乘刚也。恒不死，中未亡也。

### 白话文解释

六五：卜问疾病，长时间内不会死去。

《象辞》说：六五爻辞讲患病，因为六五阴爻居于九四阳爻之上，犯了柔乘刚之象。“长时间不会死去”，因为六五之爻居于上卦中位，正当不死之象。

### 北宋易学家 邵雍 解

凶：得此爻者，多心事，有灾难，或有腹疾。做官的多洋洋得意，持恩固宠，难免有失。

### 台湾国学大儒 傅佩荣 解

- 时运：性格柔弱，难以振作。
- 财运：用人不当，致生损失。
- 家宅：恐被侵占。
- 身体：带病延年。

### 六五变卦：雷地豫 变卦 泽地萃

六五爻动变得[第45卦：泽地萃](./e89083cui_cn.md)。

这个卦是异卦（下坤上兑）相叠。

坤为地、为顺；兑为泽、为水。

泽泛滥淹没大地，人众多相互斗争。

萃，聚集、团结。

## 上六爻详解
### 上六爻辞

上六。冥豫，成有渝，无咎。

象曰：冥豫在上，何可长也。

### 白话文解释

上六：末日将至尚且享乐，已成之事也将毁败。

《象辞》说：末日将至尚且享乐，以此居上位，怎能长久。

### 北宋易学家 邵雍 解

平：得此爻者，因为心气太高，会有诉争之扰，但若能改过自新，心态平和，则不会有过失。做官的会有贪污之嫌，须明哲保身。

### 台湾国学大儒 傅佩荣 解

- 时运：冬去春来，奋发有为。
- 财运：改旧从新，将可获利。
- 家宅：迁居改造为宜。
- 身体：应有转机。
